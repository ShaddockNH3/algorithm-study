# 算法学习笔记

这里是算法与数据结构学习记录的整理版。文档按时间和学习阶段分为三部分，阅读顺序是：

1. [past](/past/)
2. [Go 代码随想录](/go-code-thoughts/)
3. [C++ 代码随想录](/cpp-code-thoughts/)

这些文档保留了原始学习过程中的思路、错因、调试痕迹和代码实现，同时统一为更适合阅读和部署的 Markdown 格式。

## 从哪里开始

如果只是想按时间看学习轨迹，可以从 [past](/past/) 开始。这里保留了早期草稿、课程笔记、Hot100 以及最初的 C++ 代码随想录记录，更像一份原始学习档案。

如果想看当前最完整的主线，可以直接进入 [Go 代码随想录](/go-code-thoughts/)。这一部分按专题整理，从数组、链表、哈希、字符串、栈与队列，一直到动态规划和图论。

如果想对照 C++ 版本的阶段性复盘，可以进入 [C++ 代码随想录](/cpp-code-thoughts/)。这一部分主要整理到树章节，适合和 Go 版本互相参照。
