# 1.反转字符串

[344. 反转字符串 - 力扣（LeetCode）][link-1]

```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        int left=0,right=s.size()-1;
        while(left<right){
            swap(s[left++],s[right--]);
        }
    }
};
```
