# 283 移动零

## 代码

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow=0,fast=0;
        for(;fast<nums.size();fast++){
            swap(nums[slow],nums[fast]);
            if(nums[slow]!=0){
                slow++;
            }
        }
    }
};
```
