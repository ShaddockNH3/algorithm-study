# 541 反转字符串 II

## 代码

```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        int n=s.length();
        for (int i = 0; i < s.size(); i += 2 * k) {
            reverse(s.begin() + i, s.begin() + min(i + k, n));
        }
        return s;
    }
};
```
