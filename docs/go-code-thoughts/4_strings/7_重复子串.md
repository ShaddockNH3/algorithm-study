# 重复的子字符串：拼接字符串判断周期

重复子字符串可以用一个很巧妙的字符串拼接方法判断。

## 459 重复的子字符串

[题目链接](https://leetcode.cn/problems/repeated-substring-pattern/)

### 459 题意

给定一个非空字符串 `s`，
检查它是否可以由某个子串重复多次构成。

## 拼接思路

如果 `s` 可以由某个子串重复构成，
那么把 `s` 拼接成 `s + s` 后，
中间部分一定还能找到原字符串 `s`。

具体做法：

1. 构造 `t := s + s`。
2. 去掉 `t` 的第一个字符和最后一个字符。
3. 判断剩余部分是否包含 `s`。

为什么要去掉首尾字符：

- 不去掉时，`s+s` 一定包含原来的 `s`。
- 去掉后还能包含 `s`，说明 `s` 存在周期结构。

## 459 代码

```go
func repeatedSubstringPattern(s string) bool {
    t := s + s
    t = t[1 : len(t)-1]
    return strings.Contains(t, s)
}
```

## 复杂度

- 时间复杂度：与 `strings.Contains` 的实现有关，通常可视作 `O(n)`
- 空间复杂度：`O(n)`

## 本题总结

这道题的核心是利用周期字符串的性质。

如果一个字符串由重复子串构成，
那么它在双倍拼接并去掉首尾字符后，仍然能再次出现。
