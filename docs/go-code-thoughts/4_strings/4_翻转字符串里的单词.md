# 翻转字符串里的单词：清理空格与两次反转

翻转字符串里的单词需要同时处理两个问题：

- 单词顺序要反转。
- 多余空格要清理。

经典做法是三步：

1. 清理字符串中的多余空格。
2. 整体反转字符串。
3. 再逐个反转每个单词。

## 151 翻转字符串里的单词

[题目链接](https://leetcode.cn/problems/reverse-words-in-a-string/)

### 151 题意

给定一个字符串，逐个翻转字符串中的每个单词。

示例 1：

```text
输入："the sky is blue"
输出："blue is sky the"
```

示例 2：

```text
输入：" hello world! "
输出："world! hello"
```

示例 3：

```text
输入："a good example"
输出："example good a"
```

需要移除首尾多余空格，并把单词间多个空格压缩成一个空格。

## 三步思路

### 第一步：清理空格

先使用 `strings.TrimSpace` 去掉首尾空格。

之后使用快慢指针压缩中间多余空格：

- `fast` 负责遍历原始字符。
- `slow` 负责构建清理后的有效部分。

当遇到连续空格时，只保留第一个空格。

### 第二步：整体反转

清理后，先把整个字符串反转。

例如：

```text
a good example
```

整体反转后得到：

```text
elpmaxe doog a
```

这时单词顺序已经反过来了，但每个单词内部也是反的。

### 第三步：局部反转单词

再以空格为边界，逐个反转每个单词。

```text
elpmaxe -> example
doog -> good
a -> a
```

最终得到：

```text
example good a
```

## 151 代码

```go
import "strings"

func reverseBytes(b []byte, left int, right int) {
    for left < right {
        b[left], b[right] = b[right], b[left]
        left++
        right--
    }
}

func reverseWords(s string) string {
    s = strings.TrimSpace(s)
    if s == "" {
        return ""
    }

    b := []byte(s)

    slow, fast := 0, 0
    for fast < len(b) {
        if b[fast] == ' ' && b[fast-1] == ' ' {
            fast++
            continue
        }

        b[slow] = b[fast]
        slow++
        fast++
    }

    b = b[:slow]

    reverseBytes(b, 0, len(b)-1)

    wordStart := 0
    for i := 0; i <= len(b); i++ {
        if i == len(b) || b[i] == ' ' {
            reverseBytes(b, wordStart, i-1)
            wordStart = i + 1
        }
    }

    return string(b)
}
```

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

Go 字符串不可变，转换成 `[]byte` 时需要创建副本。
之后的清理和反转都在这个副本上原地完成。

## 本题总结

这道题不要直接尝试移动单词。

更稳定的思路是：

1. 先把空格处理成标准格式。
2. 整体反转，让单词顺序正确。
3. 局部反转，让单词内部恢复正确。

这个方法把复杂的单词移动问题，拆成了多个简单的反转操作。
