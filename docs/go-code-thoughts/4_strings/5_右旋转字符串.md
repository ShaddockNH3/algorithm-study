# 右旋转字符串：三次翻转

右旋转字符串可以用三次翻转完成。
这个方法适合练习原地修改和整体翻转、局部翻转的组合。

## 卡码网 右旋字符串

[题目链接](https://kamacoder.com/problempage.php?pid=1065)

### 题意

字符串的右旋转操作，是把字符串尾部的若干个字符移动到字符串前面。

给定一个字符串 `s` 和一个正整数 `k`，
将字符串后面 `k` 个字符移到字符串前面。

示例：

```text
输入：abcdefg, k = 2
输出：fgabcde
```

## 三次翻转思路

以 `abcdefg` 右旋 `2` 位为例，目标是 `fgabcde`。

### 第一步：整体翻转

先反转整个字符串：

```text
abcdefg -> gfedcba
```

### 第二步：翻转前 k 个字符

此时前 `k` 个字符是 `gf`。

将它们反转：

```text
gfedcba -> fgedcba
```

### 第三步：翻转剩余字符

再翻转后面的 `n-k` 个字符：

```text
fgedcba -> fgabcde
```

这样就完成了右旋转。

三次翻转的本质是：

1. 整体翻转先把尾部字符移动到前面。
2. 局部翻转再修正前后两段内部的顺序。

## 代码

```go
package main

import "fmt"

func reverseString(s []byte, left int, right int) {
    for left < right {
        s[left], s[right] = s[right], s[left]
        left++
        right--
    }
}

func solve() {
    var k int
    var str string

    fmt.Scan(&k)
    fmt.Scan(&str)

    b := []byte(str)
    n := len(b)
    k %= n

    reverseString(b, 0, n-1)
    reverseString(b, 0, k-1)
    reverseString(b, k, n-1)

    fmt.Println(string(b))
}

func main() {
    solve()
}
```

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

这里不计 Go 字符串转 `[]byte` 的输入转换成本。
核心翻转过程只使用了常数个变量。

## 本题总结

右旋转可以理解为把字符串分成两段：

```text
前半段 A + 后半段 B
```

右旋后需要得到：

```text
B + A
```

三次翻转正好完成这个变化：

```text
reverse(A + B) -> reverse(B) + reverse(A)
reverse(reverse(B)) + reverse(reverse(A)) -> B + A
```

这个思路也可以迁移到数组旋转问题中。
