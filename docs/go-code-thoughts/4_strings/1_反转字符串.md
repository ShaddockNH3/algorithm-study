# 反转字符串：首尾双指针交换

反转字符串是双指针最直观的应用之一。
题目要求原地修改，因此不能创建额外数组保存结果。

## 344 反转字符串

[题目链接](https://leetcode.cn/problems/reverse-string/)

### 344 题意

编写一个函数，将输入的字符串反转。

输入以字符数组 `s` 的形式给出。

要求：

- 原地修改输入数组。
- 使用 `O(1)` 额外空间。

## 双指针思路

因为只能原地修改，可以使用首尾双指针。

1. `left` 指向数组开头。
2. `right` 指向数组末尾。
3. 交换 `s[left]` 和 `s[right]`。
4. `left` 右移，`right` 左移。
5. 当 `left >= right` 时结束。

这样每次都交换对称位置上的字符。

## 344 代码

```go
func reverseString(s []byte) {
    left, right := 0, len(s)-1

    for left < right {
        s[left], s[right] = s[right], s[left]
        left++
        right--
    }
}
```

## 循环条件说明

这里使用 `left < right`。

如果数组长度为奇数，中间元素不需要和自己交换。
因此 `left < right` 是更常见的写法。

使用 `left <= right` 也能得到正确结果，
只是奇数长度时会多一次自己和自己的交换。

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

虽然只交换了数组的一半位置，但时间复杂度仍然记为 `O(n)`。

## 本题总结

当题目要求原地反转数组或字符串时，
首尾双指针通常是最直接的思路。

它的核心动作只有两个：

1. 交换左右指针指向的元素。
2. 指针向中间靠拢。
