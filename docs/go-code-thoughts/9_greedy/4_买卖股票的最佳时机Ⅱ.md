# 122 买卖股票的最佳时机 II

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/)

## 题目描述

给你一个整数数组 `prices`，
其中 `prices[i]` 表示某支股票第 `i` 天的价格。

在每一天，
你可以决定是否购买或出售股票。

任何时候最多只能持有一股股票。
也可以先购买，然后在同一天出售。

返回能够获得的最大利润。

## 思路一：寻找波峰和波谷

一个很直观的思路是把价格曲线看成山脉。

目标是在谷底买入，
在后面的波峰卖出。

步骤是：

1. 找到一个价格不再下降的位置，作为谷底。
2. 从谷底开始继续向右，找到一个价格不再上升的位置，作为波峰。
3. 累加 `peak - valley`。
4. 从波峰后继续寻找下一段上涨区间。

## 波峰波谷代码

```go
func maxProfitPeakValley(prices []int) int {
    if len(prices) <= 1 {
        return 0
    }

    maxProfit := 0
    i := 0

    for i < len(prices)-1 {
        for i < len(prices)-1 && prices[i] >= prices[i+1] {
            i++
        }
        valley := prices[i]

        for i < len(prices)-1 && prices[i] <= prices[i+1] {
            i++
        }
        peak := prices[i]

        maxProfit += peak - valley
    }

    return maxProfit
}
```

这个方法直观，
但代码需要小心控制指针和边界。

## 思路二：累加所有正收益

更简洁的贪心思路是：
只要今天价格比昨天高，
就把这一天的正收益拿到。

也就是累加所有：

```go
prices[i] - prices[i-1]
```

前提是这个差值大于 0。

连续上涨区间的每日正收益之和，
等于在上涨起点买入、上涨终点卖出的总收益。

例如 `1 -> 3 -> 5`：

```text
(3 - 1) + (5 - 3) = 5 - 1
```

所以不需要显式找完整的波峰和波谷。

## 每日收益代码

```go
func maxProfit(prices []int) int {
    maxProfit := 0

    for i := 1; i < len(prices); i++ {
        if prices[i] > prices[i-1] {
            maxProfit += prices[i] - prices[i-1]
        }
    }

    return maxProfit
}
```

## 两种思路对比

波峰波谷法：
直观模拟低买高卖，
但代码更复杂。

每日收益法：
把完整上涨区间拆成每天的小收益，
代码更短，也更不容易出错。

## 本篇总结

这道题可以把所有上涨段的收益都吃掉。

只要 `prices[i] > prices[i-1]`，
就把这一天的利润加入答案。

本质上，
连续上涨区间的逐日利润之和，
和整段低买高卖的利润完全等价。
