# 860 柠檬水找零

[题目链接](https://leetcode.cn/problems/lemonade-change/)

## 题目描述

每杯柠檬水售价 5 美元。

顾客按顺序购买，
每人只买一杯，
并支付 5 美元、10 美元或 20 美元。

一开始手里没有零钱。

如果能给每位顾客正确找零，返回 `true`；
否则返回 `false`。

## 贪心策略

收到 5 美元时，
不需要找零，只需要增加一张 5 美元。

收到 10 美元时，
必须找一张 5 美元。
如果没有 5 美元，直接失败。

收到 20 美元时，
需要找 15 美元。
有两种方案：

- 一张 10 美元加一张 5 美元。
- 三张 5 美元。

应该优先使用第一种方案。

原因是 5 美元更通用：
它既可以给 10 美元找零，
也可以参与给 20 美元找零。

10 美元只能用于给 20 美元找零。
所以要优先消耗用途更单一的 10 美元，
尽量保留 5 美元。

## 状态优化

最初可以想到用一个数组保存所有收到的钱。

但实际上 20 美元不会被用作零钱，
真正需要记录的只有：

- 5 美元有多少张。
- 10 美元有多少张。

所以用两个变量 `five` 和 `ten` 就够了。

## 最终代码

```go
func lemonadeChange(bills []int) bool {
    five := 0
    ten := 0

    for _, bill := range bills {
        if bill == 5 {
            five++
        } else if bill == 10 {
            if five > 0 {
                five--
                ten++
            } else {
                return false
            }
        } else {
            if ten > 0 && five > 0 {
                ten--
                five--
            } else if five >= 3 {
                five -= 3
            } else {
                return false
            }
        }
    }

    return true
}
```

## 本篇总结

这道题的贪心点在收到 20 美元时。

优先用 `10 + 5` 找零，
是为了尽量保留更通用的 5 美元。

状态上只需要记录 5 美元和 10 美元的数量，
不需要保存所有钞票。
