# 452 用最少数量的箭引爆气球

[题目链接](https://leetcode.cn/problems/minimum-number-of-arrows-to-burst-balloons/)

## 题目描述

有一些球形气球贴在一堵用 XY 平面表示的墙面上。

整数数组 `points` 记录这些气球，
其中 `points[i] = [xstart, xend]`
表示气球水平直径的开始和结束坐标。

一支箭可以沿着 x 轴从某个点垂直射出。
如果射箭位置 `x` 满足：

```text
xstart <= x <= xend
```

则这个气球会被引爆。

返回引爆所有气球所必须射出的最小弓箭数。

## 排序是基础

无论从射箭角度还是区间角度理解，
第一步都要排序。

这里按左边界从小到大排序：

```go
sort.Slice(points, func(i, j int) bool {
    return points[i][0] < points[j][0]
})
```

排序之后，
就可以从左到右处理所有区间。

## 思路一：维护箭的位置

至少需要一支箭引爆第一个气球。

为了尽可能覆盖后面的气球，
可以先把箭的位置放在第一个气球的右边界。

用 `arrowPosition` 记录当前这支箭还能覆盖的最右位置。

遍历后续气球时：

- 如果当前气球左边界大于 `arrowPosition`，说明需要新箭。
- 否则当前箭能引爆它，但要把箭的位置收紧到更小的右边界。

收紧右边界是为了保证这支箭仍然能穿过所有已经重叠的气球。

## 箭位置代码

```go
import "sort"

func findMinArrowShotsShooter(points [][]int) int {
    if len(points) == 0 {
        return 0
    }

    sort.Slice(points, func(i, j int) bool {
        return points[i][0] < points[j][0]
    })

    arrows := 1
    arrowPosition := points[0][1]

    for i := 1; i < len(points); i++ {
        if points[i][0] > arrowPosition {
            arrows++
            arrowPosition = points[i][1]
        } else {
            if points[i][1] < arrowPosition {
                arrowPosition = points[i][1]
            }
        }
    }

    return arrows
}
```

## 思路二：维护重叠区间

也可以把问题看成区间合并。

目标是找到最少的不相交重叠区间组。
每一组重叠区间可以用一支箭解决。

用 `rightBoundary` 维护当前重叠区域的最小右边界。

遍历后续区间：

- 如果新区间左边界大于 `rightBoundary`，说明和当前组无交集，需要新箭。
- 否则说明仍然重叠，更新 `rightBoundary` 为更小的右边界。

## 重叠区间代码

```go
import "sort"

func findMinArrowShots(points [][]int) int {
    if len(points) == 0 {
        return 0
    }

    sort.Slice(points, func(i, j int) bool {
        return points[i][0] < points[j][0]
    })

    arrows := 1
    rightBoundary := points[0][1]

    for i := 1; i < len(points); i++ {
        if points[i][0] > rightBoundary {
            arrows++
            rightBoundary = points[i][1]
        } else {
            if points[i][1] < rightBoundary {
                rightBoundary = points[i][1]
            }
        }
    }

    return arrows
}
```

## 本篇总结

两种叙述最后是同一套逻辑。

`arrowPosition` 和 `rightBoundary` 都表示当前这组重叠气球
可以共同引爆的位置范围。

一旦新区间左边界超过这个范围，
就必须增加一支箭，
并开启新的重叠区间组。
