# 56 合并区间

[题目链接](https://leetcode.cn/problems/merge-intervals/)

## 题目描述

以数组 `intervals` 表示若干个区间的集合，
其中单个区间为 `intervals[i] = [starti, endi]`。

请合并所有重叠的区间，
并返回一个不重叠的区间数组。

返回数组需要恰好覆盖输入中的所有区间。

## 排序是基础

区间问题通常先排序。

这里按左边界从小到大排序：

```go
sort.Slice(intervals, func(i, j int) bool {
    return intervals[i][0] < intervals[j][0]
})
```

排序后，
可能和当前区间重叠的区间都会相邻出现，
可以从左到右顺序处理。

## 思路一：使用 start 和 end

用两个变量 `start` 和 `end`
维护当前正在合并的大区间。

初始时，
当前大区间就是排序后的第一个区间。

遍历后续区间时：

- 如果新区间左边界大于 `end`，说明不重叠。
- 如果重叠，就更新 `end = max(end, intervals[i][1])`。

遇到不重叠区间时，
要先把当前 `[start, end]` 放入结果集，
再开启新的合并区间。

循环结束后，
别忘了把最后一个合并区间放入结果集。

## start 和 end 代码

```go
import "sort"

func mergeVars(intervals [][]int) [][]int {
    if len(intervals) <= 1 {
        return intervals
    }

    sort.Slice(intervals, func(i, j int) bool {
        return intervals[i][0] < intervals[j][0]
    })

    merged := make([][]int, 0)
    start := intervals[0][0]
    end := intervals[0][1]

    for i := 1; i < len(intervals); i++ {
        if intervals[i][0] > end {
            merged = append(merged, []int{start, end})
            start = intervals[i][0]
            end = intervals[i][1]
        } else {
            if intervals[i][1] > end {
                end = intervals[i][1]
            }
        }
    }

    merged = append(merged, []int{start, end})
    return merged
}
```

## 思路二：直接操作结果集

更稳的写法是直接维护结果集 `merged`。

先把第一个区间放入结果集。

之后每次拿当前区间和结果集最后一个区间比较：

- 如果重叠，更新结果集最后一个区间的右边界。
- 如果不重叠，把当前区间追加到结果集。

这样不会忘记最后一个区间，
因为结果集始终保持最新状态。

## 结果集代码

```go
import "sort"

func merge(intervals [][]int) [][]int {
    if len(intervals) <= 1 {
        return intervals
    }

    sort.Slice(intervals, func(i, j int) bool {
        return intervals[i][0] < intervals[j][0]
    })

    merged := [][]int{intervals[0]}

    for i := 1; i < len(intervals); i++ {
        lastInterval := merged[len(merged)-1]

        if intervals[i][0] <= lastInterval[1] {
            if intervals[i][1] > lastInterval[1] {
                merged[len(merged)-1][1] = intervals[i][1]
            }
        } else {
            merged = append(merged, intervals[i])
        }
    }

    return merged
}
```

## 本篇总结

合并区间的核心是先按左边界排序。

排序后只需要关注当前区间和已经合并结果的最后一个区间。

可以用临时变量维护当前合并区间，
也可以直接更新结果集最后一个区间。

后者通常更不容易漏掉最后一个区间。
