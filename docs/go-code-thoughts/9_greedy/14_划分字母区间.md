# 763 划分字母区间

[题目链接](https://leetcode.cn/problems/partition-labels/)

## 题目描述

给你一个字符串 `s`。

需要把这个字符串划分为尽可能多的片段，
并且同一字母最多出现在一个片段中。

划分结果按顺序连接后，
仍然应该得到原字符串 `s`。

返回每个字符串片段的长度。

## 贪心思路

题目要求片段尽可能多，
也就是每个片段要尽可能短。

一个片段什么时候可以安全切开？

当当前片段中出现过的所有字母，
它们在整个字符串中的最后一次出现位置，
都已经被当前片段包含时，
就可以切开。

所以需要先知道每个字母最后出现的位置。

## 两次遍历

第一次遍历字符串，
用 `lastPos` 记录每个字母最后一次出现的位置。

第二次遍历时，
维护当前片段的起点 `start` 和右边界 `end`。

每遇到一个字母 `c`，
就用它的最后位置更新 `end`：

```go
if lastPos[c-'a'] > end {
    end = lastPos[c-'a']
}
```

当遍历指针 `i == end` 时，
说明当前片段中所有字母的最后出现位置都被覆盖了。
此时可以切出一段。

## 最终代码

```go
func partitionLabels(s string) (partition []int) {
    lastPos := [26]int{}

    for i, c := range s {
        lastPos[c-'a'] = i
    }

    start, end := 0, 0
    for i, c := range s {
        if lastPos[c-'a'] > end {
            end = lastPos[c-'a']
        }

        if i == end {
            partition = append(partition, end-start+1)
            start = end + 1
        }
    }

    return
}
```

## 本篇总结

划分字母区间的关键是找到安全切割点。

先预处理每个字母最后出现的位置。

扫描当前片段时不断扩张右边界。
当当前位置到达右边界时，
说明这一段已经覆盖其中所有字母的最后出现位置，
可以立即切开。
