# 406 根据身高重建队列

[题目链接](https://leetcode.cn/problems/queue-reconstruction-by-height/)

## 题目描述

假设有一群人站成一个打乱顺序的队列。

数组 `people` 表示这些人的属性。
每个 `people[i] = [hi, ki]` 表示：

- 第 `i` 个人的身高为 `hi`。
- 他前面正好有 `ki` 个身高大于或等于 `hi` 的人。

请重新构造并返回队列。

## 思路一：从矮到高，预留空位

最初的思路是先处理矮个子。

排序规则：

- 身高从小到大。
- 身高相同，`k` 从小到大。

因为先处理矮个子，
所有还没有放入队列的人都比当前人高或一样高。

所以当前人前面的空位，
将来都会被更高或一样高的人填上。

放置当前人时，
只需要找到第 `k` 个空位，
把他放进去即可。

## 预留空位代码

```go
import "sort"

func reconstructQueueFromShortest(people [][]int) [][]int {
    sort.Slice(people, func(i, j int) bool {
        if people[i][0] != people[j][0] {
            return people[i][0] < people[j][0]
        }
        return people[i][1] < people[j][1]
    })

    queue := make([][]int, len(people))
    for i := range queue {
        queue[i] = []int{-1, -1}
    }

    for _, person := range people {
        k := person[1]
        emptySpaces := 0

        for i := 0; i < len(queue); i++ {
            if queue[i][0] == -1 {
                if emptySpaces == k {
                    queue[i] = person
                    break
                }
                emptySpaces++
            }
        }
    }

    return queue
}
```

这个思路能解释得通，
但实现需要维护空位，
代码相对绕。

## 思路二：从高到矮，按 k 插入

经典做法是先处理高个子。

排序规则：

- 身高从高到低。
- 身高相同，`k` 从小到大。

当处理某个人时，
已经进入队列的人都比他高或一样高。

所以他的 `k` 值就直接表示：
他应该插入到当前队列的第 `k` 个位置。

后续插入的矮个子不会影响他的 `k` 计数，
因为矮个子不属于身高大于或等于他的人。

## 按 k 插入代码

```go
import "sort"

func reconstructQueue(people [][]int) [][]int {
    sort.Slice(people, func(i, j int) bool {
        if people[i][0] != people[j][0] {
            return people[i][0] > people[j][0]
        }
        return people[i][1] < people[j][1]
    })

    queue := make([][]int, 0, len(people))

    for _, person := range people {
        k := person[1]

        queue = append(queue, []int{})
        copy(queue[k+1:], queue[k:])
        queue[k] = person
    }

    return queue
}
```

## 本篇总结

这道题可以从矮到高预留空位，
也可以从高到矮按 `k` 插入。

更常用的是从高到矮：
因为已经放入队列的人都满足身高大于或等于当前人，
所以 `k` 可以直接作为插入位置。

后续矮个子的插入不会改变高个子的计数。
