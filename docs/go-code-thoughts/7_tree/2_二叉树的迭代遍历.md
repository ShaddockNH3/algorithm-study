# 二叉树的迭代遍历：用栈模拟递归

二叉树的迭代遍历，本质上是用显式栈模拟递归调用栈。

递归由系统调用栈维护节点访问顺序。
迭代写法则需要自己控制入栈、出栈和访问时机。

## 144 二叉树的前序遍历

[题目链接](https://leetcode.cn/problems/binary-tree-preorder-traversal/)

### 144 迭代思路

前序遍历顺序是：

```text
中 -> 左 -> 右
```

使用栈时：

1. 先将根节点入栈。
2. 每次弹出栈顶节点并处理。
3. 因为栈是后进先出，所以先压入右孩子。
4. 再压入左孩子。

这样左孩子会比右孩子更先出栈。

### 144 代码

```go
func preorderTraversal(root *TreeNode) []int {
    if root == nil {
        return []int{}
    }

    stack := []*TreeNode{root}
    ans := []int{}

    for len(stack) > 0 {
        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]

        ans = append(ans, node.Val)

        if node.Right != nil {
            stack = append(stack, node.Right)
        }
        if node.Left != nil {
            stack = append(stack, node.Left)
        }
    }

    return ans
}
```

## 94 二叉树的中序遍历

[题目链接](https://leetcode.cn/problems/binary-tree-inorder-traversal/)

### 94 迭代思路

中序遍历顺序是：

```text
左 -> 中 -> 右
```

中序遍历不能弹出节点就立刻处理。
必须先一路向左，把左侧路径上的节点都压入栈。

处理步骤：

1. 用 `curr` 从根节点开始。
2. 当 `curr != nil` 时，不断向左走，并把节点入栈。
3. 左边走到尽头后，从栈中弹出节点。
4. 处理这个节点。
5. 让 `curr` 指向它的右孩子。
6. 重复上述过程。

### 94 代码

```go
func inorderTraversal(root *TreeNode) []int {
    ans := []int{}
    stack := []*TreeNode{}
    curr := root

    for curr != nil || len(stack) > 0 {
        for curr != nil {
            stack = append(stack, curr)
            curr = curr.Left
        }

        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        ans = append(ans, node.Val)
        curr = node.Right
    }

    return ans
}
```

## 145 二叉树的后序遍历

[题目链接](https://leetcode.cn/problems/binary-tree-postorder-traversal/)

### 145 迭代思路

后序遍历顺序是：

```text
左 -> 右 -> 中
```

直接模拟后序比较绕。
可以先构造一个相反顺序：

```text
中 -> 右 -> 左
```

再整体反转，得到：

```text
左 -> 右 -> 中
```

这个过程和前序遍历很像。
前序是先压右，再压左，从而得到中左右。

这里要得到中右左，所以改成先压左，再压右。

### 145 代码

```go
import "slices"

func postorderTraversal(root *TreeNode) []int {
    if root == nil {
        return []int{}
    }

    ans := []int{}
    stack := []*TreeNode{root}

    for len(stack) > 0 {
        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]

        ans = append(ans, node.Val)

        if node.Left != nil {
            stack = append(stack, node.Left)
        }
        if node.Right != nil {
            stack = append(stack, node.Right)
        }
    }

    slices.Reverse(ans)
    return ans
}
```

## Go 原地操作注意点

`slices.Reverse(ans)` 和 `sort.Ints(ans)` 这类函数是原地操作。

它们会直接修改传入的切片，并且没有返回值。

正确写法是：

```go
slices.Reverse(ans)
return ans
```

不要写成试图接收返回值的形式。

## 本篇总结

前序遍历最适合直接用栈模拟。

中序遍历需要使用 `curr` 一路向左，
因为访问当前节点必须发生在左子树之后。

后序遍历可以通过中右左再反转得到，
从而复用前序遍历的思路。
