# 对称二叉树：判断左右子树是否互为镜像

对称二叉树的核心是判断根节点的左子树和右子树是否互为镜像。

镜像关系不是简单地比较左对左、右对右，
而是需要交叉比较：

- 左子树的左侧，对应右子树的右侧。
- 左子树的右侧，对应右子树的左侧。

## 101 对称二叉树

[题目链接](https://leetcode.cn/problems/symmetric-tree/)

### 101 题意

给定一棵二叉树的根节点 `root`，检查它是否轴对称。

示例：

```text
      1
     / \
    2   2
   / \ / \
  3  4 4  3
```

这是一棵对称二叉树，返回 `true`。

## 方法一：双队列迭代

### 双队列思路

同时对左子树和右子树做层次遍历。

左子树按正常左右顺序入队，
右子树按镜像顺序入队。

每次取出两个节点进行比较：

1. 两个节点都为空，继续。
2. 一个为空一个不为空，返回 `false`。
3. 两个节点值不同，返回 `false`。
4. 节点值相同，则继续比较它们的孩子。

### 双队列代码

```go
func isSymmetric(root *TreeNode) bool {
    if root == nil {
        return true
    }

    queueLeft := []*TreeNode{root.Left}
    queueRight := []*TreeNode{root.Right}

    for len(queueLeft) > 0 && len(queueRight) > 0 {
        if len(queueLeft) != len(queueRight) {
            return false
        }

        levelLen := len(queueLeft)

        for i := 0; i < levelLen; i++ {
            nodeLeft := queueLeft[0]
            queueLeft = queueLeft[1:]

            nodeRight := queueRight[0]
            queueRight = queueRight[1:]

            if nodeLeft == nil && nodeRight == nil {
                continue
            }
            if nodeLeft == nil ||
                nodeRight == nil ||
                nodeLeft.Val != nodeRight.Val {
                return false
            }

            queueLeft = append(queueLeft, nodeLeft.Left)
            queueLeft = append(queueLeft, nodeLeft.Right)

            queueRight = append(queueRight, nodeRight.Right)
            queueRight = append(queueRight, nodeRight.Left)
        }
    }

    return len(queueLeft) == len(queueRight)
}
```

## 方法二：递归镜像

### 递归思路

判断一棵树是否对称，可以转化为：

```text
root.Left 和 root.Right 是否互为镜像
```

两棵树 `p` 和 `q` 互为镜像，需要满足：

1. 根节点值相同。
2. `p.Left` 和 `q.Right` 互为镜像。
3. `p.Right` 和 `q.Left` 互为镜像。

### 递归代码

```go
func isSymmetric(root *TreeNode) bool {
    if root == nil {
        return true
    }

    return isMirror(root.Left, root.Right)
}

func isMirror(p, q *TreeNode) bool {
    if p == nil && q == nil {
        return true
    }

    if p == nil || q == nil || p.Val != q.Val {
        return false
    }

    return isMirror(p.Left, q.Right) && isMirror(p.Right, q.Left)
}
```

## 方法三：单队列迭代

### 单队列思路

单队列法把需要比较的节点成对入队。

每次从队列中取出两个节点 `p` 和 `q`：

1. 如果都为空，继续。
2. 如果只有一个为空，返回 `false`。
3. 如果值不同，返回 `false`。
4. 如果通过比较，就把下一组镜像节点入队。

入队顺序是：

```text
p.Left  和 q.Right
p.Right 和 q.Left
```

### 单队列代码

```go
func isSymmetric(root *TreeNode) bool {
    if root == nil {
        return true
    }

    queue := []*TreeNode{root.Left, root.Right}

    for len(queue) > 0 {
        p := queue[0]
        q := queue[1]
        queue = queue[2:]

        if p == nil && q == nil {
            continue
        }
        if p == nil || q == nil || p.Val != q.Val {
            return false
        }

        queue = append(queue, p.Left)
        queue = append(queue, q.Right)
        queue = append(queue, p.Right)
        queue = append(queue, q.Left)
    }

    return true
}
```

## 复杂度

三种方法的时间复杂度都是 `O(n)`。

空间复杂度：

- 递归法：`O(h)`，其中 `h` 是树高。
- 迭代法：`O(n)`，最坏情况下队列保存一层节点。

## 本题总结

对称二叉树的关键是交叉比较。

无论用递归还是迭代，都要比较：

- 左子树的左节点与右子树的右节点。
- 左子树的右节点与右子树的左节点。

这就是镜像关系和普通相等判断的区别。
