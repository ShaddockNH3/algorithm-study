# 235 二叉搜索树的最近公共祖先

[题目链接](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-search-tree/)

## 题目描述

给定一个二叉搜索树，找到该树中两个指定节点的最近公共祖先。

## 和普通二叉树的区别

上一题是普通二叉树的最近公共祖先。
因为普通二叉树没有顺序信息，站在一个节点上时，
无法提前知道 `p` 和 `q` 在左子树还是右子树。

所以普通二叉树需要先递归搜索左右子树，
再根据左右子树返回的信息判断当前节点是不是最近公共祖先。
这个过程本质上是自底向上的后序思路。

二叉搜索树不一样。
BST 有序，当前节点可以直接提供方向信息：

- 如果 `p.Val` 和 `q.Val` 都小于 `root.Val`，答案一定在左子树。
- 如果 `p.Val` 和 `q.Val` 都大于 `root.Val`，答案一定在右子树。
- 否则当前节点就是分岔点，也就是最近公共祖先。

因此这道题可以自顶向下查找，不需要把左右子树都完整搜一遍。

## 迭代思路

用 `root` 指针从根节点开始向下走。

只要 `root` 不为空，就根据 `p`、`q` 和 `root` 的值关系判断方向。

当两个目标都在当前节点左侧时，令 `root = root.Left`。

当两个目标都在当前节点右侧时，令 `root = root.Right`。

如果不满足以上两种情况，说明两条路径在当前节点分开。
也可能是其中一个节点正好就是当前节点。
无论哪种情况，当前节点都是最近公共祖先。

## 迭代代码

```go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */

func lowestCommonAncestor(root, p, q *TreeNode) *TreeNode {
    for root != nil {
        if p.Val < root.Val && q.Val < root.Val {
            root = root.Left
        } else if p.Val > root.Val && q.Val > root.Val {
            root = root.Right
        } else {
            return root
        }
    }
    return nil
}
```

## 递归写法

同样的逻辑也可以写成递归。
这里递归只是把循环移动节点的过程换成函数调用，
判断条件没有变化。

```go
func lowestCommonAncestor_Recursive(root, p, q *TreeNode) *TreeNode {
    if p.Val < root.Val && q.Val < root.Val {
        return lowestCommonAncestor_Recursive(root.Left, p, q)
    }
    if p.Val > root.Val && q.Val > root.Val {
        return lowestCommonAncestor_Recursive(root.Right, p, q)
    }
    return root
}
```

## 本篇总结

普通二叉树的最近公共祖先依赖左右子树的返回结果，
所以更像后序遍历。

二叉搜索树可以利用节点值的大小关系，从根节点开始不断缩小范围。
找到第一个让 `p` 和 `q` 分到不同方向的节点时，
这个节点就是最近公共祖先。
