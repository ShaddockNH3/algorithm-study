# 层序遍历专题二：填充每个节点的右侧指针

这道题仍然可以用层序遍历解决。

区别是遍历每一层时，不只是收集节点值，
还要把当前节点的 `Next` 指针指向同一层的右侧节点。

## 116 填充每个节点的下一个右侧节点指针

[题目链接](https://leetcode.cn/problems/populating-next-right-pointers-in-each-node/)

### 116 题意

给定一棵完美二叉树。

完美二叉树满足：

- 所有叶子节点都在同一层。
- 每个父节点都有两个子节点。

节点结构包含 `Next` 指针：

```text
struct Node {
  int val;
  Node *left;
  Node *right;
  Node *next;
}
```

要求填充每个节点的 `Next` 指针，
让它指向同一层的下一个右侧节点。

如果找不到下一个右侧节点，则将 `Next` 设置为 `nil`。

初始状态下，所有 `Next` 指针都为 `nil`。

## 层序遍历思路

层序遍历时，每次都能知道当前层的节点数量 `levelSize`。

遍历当前层时：

1. 依次弹出当前层节点。
2. 把左右孩子加入队列。
3. 如果当前节点不是本层最后一个节点，
   就让它的 `Next` 指向队列中的下一个节点。
4. 如果当前节点是本层最后一个节点，`Next` 保持 `nil`。

## 116 代码

```go
func connect(root *Node) *Node {
    if root == nil {
        return root
    }

    queue := []*Node{root}

    for len(queue) > 0 {
        levelSize := len(queue)

        for i := 0; i < levelSize; i++ {
            node := queue[0]
            queue = queue[1:]

            if node.Left != nil {
                queue = append(queue, node.Left)
            }
            if node.Right != nil {
                queue = append(queue, node.Right)
            }

            if i != levelSize-1 {
                node.Next = queue[0]
            }
        }
    }

    return root
}
```

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

队列最多保存一层节点。

## 本题总结

这道题依然是层序遍历模板。

关键是遍历每一层时，利用 `levelSize` 判断当前节点是否为本层最后一个。
如果不是最后一个，就可以连接到同层右侧节点。
