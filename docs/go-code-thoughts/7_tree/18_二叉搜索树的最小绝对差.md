# 二叉搜索树的最小绝对差：中序遍历相邻比较

二叉搜索树的中序遍历会得到严格递增序列。

在有序序列中，任意两数的最小差值一定出现在相邻元素之间。

因此，这道题可以转化为：

```text
对 BST 做中序遍历，然后比较相邻节点值
```

## 530 二叉搜索树的最小绝对差

[题目链接](https://leetcode.cn/problems/minimum-absolute-difference-in-bst/)

### 530 题意

给定一棵二叉搜索树的根节点 `root`，
返回树中任意两个不同节点值之间的最小差值。

差值是正数，等于两值之差的绝对值。

## 方法一：中序遍历保存数组

### 两步思路

第一步：对 BST 做中序遍历，得到有序数组。

第二步：遍历有序数组，计算相邻元素差值的最小值。

这个方法直观，容易理解。

### 两步法代码

```go
func getMinimumDifference(root *TreeNode) int {
    res := []int{}
    inorderIter(root, &res)

    minNum := abs(res[1] - res[0])
    for i := 2; i < len(res); i++ {
        minNum = min(minNum, abs(res[i]-res[i-1]))
    }

    return minNum
}

func inorderIter(root *TreeNode, result *[]int) {
    stack := []*TreeNode{}
    curr := root

    for curr != nil || len(stack) > 0 {
        for curr != nil {
            stack = append(stack, curr)
            curr = curr.Left
        }

        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]

        *result = append(*result, node.Val)
        curr = node.Right
    }
}
```

### 递归中序写法

```go
func inorder(node *TreeNode, result *[]int) {
    if node == nil {
        return
    }

    inorder(node.Left, result)
    *result = append(*result, node.Val)
    inorder(node.Right, result)
}
```

## 方法二：中序遍历过程中直接计算

### 优化思路

其实不需要保存完整数组。

因为只需要比较中序遍历中的相邻节点，
所以在遍历过程中维护前一个访问值 `prev` 即可。

当访问当前节点时：

1. 如果已经有前一个值，就计算 `node.Val - prev`。
2. 更新最小差值。
3. 将 `prev` 更新为当前节点值。

### 优化代码

```go
import "math"

func getMinimumDifferenceOptimized(root *TreeNode) int {
    prev := -1
    minDiff := math.MaxInt32

    var inorder func(*TreeNode)
    inorder = func(node *TreeNode) {
        if node == nil {
            return
        }

        inorder(node.Left)

        if prev != -1 {
            minDiff = min(minDiff, node.Val-prev)
        }
        prev = node.Val

        inorder(node.Right)
    }

    inorder(root)
    return minDiff
}

func min(a, b int) int {
    if a < b {
        return a
    }
    return b
}
```

## 复杂度

保存数组的方法：

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

一次遍历直接计算的方法：

- 时间复杂度：`O(n)`
- 空间复杂度：`O(h)`

其中 `h` 是树高，空间主要来自递归调用栈。

## 本题总结

BST 题目要优先想到中序遍历。

这道题的关键转换是：

```text
树中任意两节点最小差
-> 中序有序数组中相邻元素最小差
```

保存完整数组更直观。
在遍历过程中维护 `prev` 更节省空间。
