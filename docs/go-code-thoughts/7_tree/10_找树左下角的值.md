# 找树左下角的值：DFS 深度记录与 BFS 层序覆盖

找树左下角的值，目标是找到最底层最左边节点的值。

可以用两种思路：

- DFS：优先向左遍历，并在发现更深层时更新答案。
- BFS：按层遍历，每层用第一个节点覆盖答案。

## 513 找树左下角的值

[题目链接](https://leetcode.cn/problems/find-bottom-left-tree-value/)

### 513 题意

给定二叉树的根节点 `root`，
找出该二叉树最底层最左边节点的值。

题目保证二叉树至少有一个节点。

## 方法一：DFS

### DFS 思路

DFS 使用两个全局变量：

- `maxDepth`：当前已经发现的最大深度。
- `resValue`：最大深度上最左节点的值。

递归时携带当前深度 `currentDepth`。

遍历顺序必须是先左后右。

每到一个节点，如果 `currentDepth > maxDepth`，
说明第一次到达了一个更深的层级。

因为 DFS 总是先访问左子树，
所以第一次到达某个深度的节点，就是该层最左边的节点。

### DFS 代码

```go
func findBottomLeftValue(root *TreeNode) int {
    maxDepth := -1
    resValue := 0

    var dfs func(node *TreeNode, currentDepth int)

    dfs = func(node *TreeNode, currentDepth int) {
        if node == nil {
            return
        }

        if currentDepth > maxDepth {
            maxDepth = currentDepth
            resValue = node.Val
        }

        dfs(node.Left, currentDepth+1)
        dfs(node.Right, currentDepth+1)
    }

    dfs(root, 0)
    return resValue
}
```

### DFS 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(h)`

其中 `h` 是树高。

## 方法二：BFS

### BFS 思路

BFS 按层遍历。

每一层都从左到右处理节点。
所以每层的第一个节点，就是该层最左边的节点。

用 `resValue` 在每层开始时记录当前层第一个节点的值。
当遍历结束时，最后一次记录的值就是最底层最左边节点的值。

### BFS 代码

```go
func findBottomLeftValueBFS(root *TreeNode) int {
    if root == nil {
        return 0
    }

    resValue := 0
    queue := []*TreeNode{root}

    for len(queue) > 0 {
        levelSize := len(queue)

        for i := 0; i < levelSize; i++ {
            node := queue[0]
            queue = queue[1:]

            if i == 0 {
                resValue = node.Val
            }

            if node.Left != nil {
                queue = append(queue, node.Left)
            }
            if node.Right != nil {
                queue = append(queue, node.Right)
            }
        }
    }

    return resValue
}
```

### BFS 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

队列最多可能保存一层节点。

## 本题总结

DFS 的关键是先左后右。
第一次到达新深度时记录的节点，就是该深度最左边节点。

BFS 的关键是按层处理。
每层第一个节点覆盖答案，最后留下的就是最底层最左节点。

这道题适合同时练习 DFS 的深度记录和 BFS 的层序思维。
