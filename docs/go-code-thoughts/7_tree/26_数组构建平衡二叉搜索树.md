# 108 将有序数组转换为二叉搜索树

[题目链接](https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/)

## 题目描述

给你一个整数数组 `nums`，
其中元素已经按升序排列。

请将其转换为一棵高度平衡二叉搜索树。

## 核心思路

题目不仅要求构造 BST，还要求高度平衡。

对于一个已经排好序的数组，
最自然的做法是每次选择中间元素作为当前子树的根节点。

这样左半部分元素进入左子树，右半部分元素进入右子树。
左右子树的节点数量尽量接近，
递归地这样处理，就能构造出平衡的 BST。

这个过程就是分治：

1. 在当前区间中选中点作为根。
2. 用左半区间构造左子树。
3. 用右半区间构造右子树。
4. 返回当前根节点。

## 递归函数的约定

`getTree(nums, left, right)` 的含义是：

用 `nums[left:right]` 这个闭区间里的元素构造一棵平衡 BST，
并返回这棵树的根节点。

如果 `left > right`，说明当前区间为空，
返回 `nil`。

如果区间不为空，就取中点：

```go
mid := left + (right-left)/2
```

然后创建当前根节点，再分别构造左右子树。

## 曾经的范围错误

一开始容易把递归范围写成这样：

```go
root.Left = getTree(nums, 0, mid-1)
root.Right = getTree(nums, mid+1, len(nums)-1)
```

这个写法的问题是，每一层递归都重新使用了整个数组的边界。
当前递归只应该处理自己负责的区间，
不能把范围重置回全局的 `0` 和 `len(nums)-1`。

正确写法应该使用当前函数收到的 `left` 和 `right`：

```go
root.Left = getTree(nums, left, mid-1)
root.Right = getTree(nums, mid+1, right)
```

这样每一层递归都只处理自己的子区间。

## 最终代码

```go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */

func sortedArrayToBST(nums []int) *TreeNode {
    return getTree(nums, 0, len(nums)-1)
}

func getTree(nums []int, left, right int) *TreeNode {
    if left > right {
        return nil
    }

    mid := left + (right-left)/2
    root := &TreeNode{Val: nums[mid]}

    root.Left = getTree(nums, left, mid-1)
    root.Right = getTree(nums, mid+1, right)

    return root
}
```

## 本篇总结

有序数组构造平衡 BST 的关键是每次取中点。

递归时一定要传递当前子区间的边界。
如果每次都回到全数组边界，
递归范围就会混乱，甚至造成重复处理和无限递归。
