# 701 二叉搜索树中的插入操作

[题目链接](https://leetcode.cn/problems/insert-into-a-binary-search-tree/)

## 题目描述

给定二叉搜索树的根节点 `root` 和要插入树中的值 `val`，
将值插入二叉搜索树。
返回插入后二叉搜索树的根节点。

输入数据保证新值和原始二叉搜索树中的任意节点值都不同。

注意，可能存在多种有效的插入方式。
只要树在插入后仍保持二叉搜索树性质即可。

## 迭代思路

插入 BST 的核心是利用它的有序性向下查找空位置。

从根节点出发：

- 如果 `val` 比当前节点大，就去右子树。
- 如果 `val` 比当前节点小，就去左子树。

当指针走到 `nil` 时，就找到了新节点应该插入的位置。

不过指针走到 `nil` 后，已经没有办法直接挂接新节点。
所以需要额外记录前一个节点，也就是插入位置的父节点。

这个版本里：

- `tmp` 负责向下查找。
- `pre` 记录 `tmp` 的父节点。

循环结束后，再根据 `val` 和 `pre.Val` 的大小关系，
决定新节点挂在 `pre.Left` 还是 `pre.Right`。

## 迭代代码

```go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func insertIntoBST(root *TreeNode, val int) *TreeNode {
    if root == nil {
        return &TreeNode{Val: val}
    }

    var pre *TreeNode
    tmp := root

    for tmp != nil {
        pre = tmp
        if tmp.Val < val {
            tmp = tmp.Right
        } else {
            tmp = tmp.Left
        }
    }

    newNode := &TreeNode{Val: val}

    if pre.Val < val {
        pre.Right = newNode
    } else {
        pre.Left = newNode
    }

    return root
}
```

## 递归思路

递归版本的关键是明确函数的含义：

`insertIntoBST(root, val)` 会把 `val` 插入到以 `root` 为根的树中，
并返回插入后的这棵树的新根节点。

如果当前节点为空，说明找到了真正的插入位置，
直接创建新节点并返回。

如果 `val` 大于当前节点值，就递归插入右子树。
递归返回后，要把新的右子树重新接回 `root.Right`。

如果 `val` 小于当前节点值，就递归插入左子树。
递归返回后，要把新的左子树重新接回 `root.Left`。

最后返回当前节点 `root`，让上一层继续连接。

## 递归代码

```go
func insertIntoBST_Recursive(root *TreeNode, val int) *TreeNode {
    if root == nil {
        return &TreeNode{Val: val}
    }

    if val > root.Val {
        root.Right = insertIntoBST_Recursive(root.Right, val)
    } else {
        root.Left = insertIntoBST_Recursive(root.Left, val)
    }

    return root
}
```

## 本篇总结

迭代法更直观，需要用 `pre` 记录父节点，
因为真正插入时 `tmp` 已经走到了空位置。

递归法更简洁，重点是让递归函数返回插入后的子树根节点，
再把这个返回值接回父节点。

两种写法都依赖 BST 的导航性质。
