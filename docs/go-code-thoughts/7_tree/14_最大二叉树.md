# 最大二叉树：递归构造与返回子树

最大二叉树是一类递归构造问题。

题目本身已经给出了递归定义：

1. 当前数组最大值作为根节点。
2. 最大值左侧数组构造左子树。
3. 最大值右侧数组构造右子树。

## 654 最大二叉树

[题目链接](https://leetcode.cn/problems/maximum-binary-tree/)

### 654 题意

给定一个不重复的整数数组 `nums`，
按照以下规则构建最大二叉树：

1. 创建根节点，值为 `nums` 中的最大值。
2. 在最大值左边的子数组上递归构建左子树。
3. 在最大值右边的子数组上递归构建右子树。

返回构建出的最大二叉树。

## 递归构造思路

递归函数 `buildTree(nums, left, right)` 负责构造区间
`[left, right]` 对应的子树，并返回这棵子树的根节点。

处理步骤：

1. 如果 `left > right`，说明区间为空，返回 `nil`。
2. 在当前区间内找到最大值下标 `maxi`。
3. 创建根节点 `root`。
4. 递归构造左区间 `[left, maxi-1]`。
5. 递归构造右区间 `[maxi+1, right]`。
6. 返回当前根节点。

这里最重要的是理解返回值。

每一次递归调用都会返回一棵已经构建好的子树。
父节点接收这个返回结果，并把它挂到自己的 `Left` 或 `Right` 上。

## 654 代码

```go
func constructMaximumBinaryTree(nums []int) *TreeNode {
    return buildTree(nums, 0, len(nums)-1)
}

func buildTree(nums []int, left int, right int) *TreeNode {
    if left > right {
        return nil
    }

    maxi := left
    for i := left + 1; i <= right; i++ {
        if nums[i] > nums[maxi] {
            maxi = i
        }
    }

    root := &TreeNode{Val: nums[maxi]}

    root.Left = buildTree(nums, left, maxi-1)
    root.Right = buildTree(nums, maxi+1, right)

    return root
}
```

## 复杂度

- 时间复杂度：最坏 `O(n^2)`
- 空间复杂度：最坏 `O(n)`

如果数组本身单调，每次最大值都在边界，
递归会退化成链状结构。

如果每次最大值比较均匀地划分区间，递归深度会更低。

## 本题总结

最大二叉树的关键是递归构造。

递归不只是向下调用，
还要关注每一层调用向上返回了什么。

在这道题中，每一层返回的是当前区间构造好的子树根节点。
父节点通过接收这些返回值，完成整棵树的组装。
