# 翻转二叉树：递归与层序遍历

翻转二叉树的核心操作是交换每个节点的左右孩子。

可以用递归深度优先处理，也可以用队列层序遍历处理。

## 226 翻转二叉树

[题目链接](https://leetcode.cn/problems/invert-binary-tree/)

### 226 题意

给定一棵二叉树的根节点 `root`，翻转这棵二叉树，
并返回其根节点。

示例输入：

```text
      4
     / \
    2   7
   / \ / \
  1  3 6  9
```

示例输出：

```text
      4
     / \
    7   2
   / \ / \
  9  6 3  1
```

## 方法一：递归

### 递归思路

递归法利用树本身的递归结构。

对于当前节点：

1. 如果节点为 `nil`，直接返回。
2. 交换当前节点的左右孩子。
3. 递归翻转左子树。
4. 递归翻转右子树。
5. 返回根节点。

### 递归代码

```go
func invertTree(root *TreeNode) *TreeNode {
    if root == nil {
        return nil
    }

    root.Left, root.Right = root.Right, root.Left

    invertTree(root.Left)
    invertTree(root.Right)

    return root
}
```

### 递归复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(h)`

其中 `h` 是树的高度，空间来自递归调用栈。

## 方法二：层序遍历

### 层序思路

也可以用队列一层一层遍历。

每取出一个节点，就交换它的左右孩子。
然后把交换后的非空孩子加入队列。

### 层序代码

```go
func invertTree(root *TreeNode) *TreeNode {
    if root == nil {
        return nil
    }

    queue := []*TreeNode{root}

    for len(queue) > 0 {
        node := queue[0]
        queue = queue[1:]

        node.Left, node.Right = node.Right, node.Left

        if node.Left != nil {
            queue = append(queue, node.Left)
        }
        if node.Right != nil {
            queue = append(queue, node.Right)
        }
    }

    return root
}
```

### 层序复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

最宽的一层可能进入队列。

## 方法对比

- 递归法：
  代码简洁，最能体现问题本质。
- 层序遍历：
  不依赖递归调用栈，流程更显式。

## 本题总结

翻转二叉树时，不需要移动节点本身。

只要访问到每个节点，并交换它的 `Left` 和 `Right` 指针，
整棵树就会被翻转。
