# 二叉树的所有路径：DFS 记录根到叶路径

二叉树的所有路径要求找出每一条从根节点到叶子节点的路径。

路径需要用 `->` 连接成字符串，例如 `1->2->5`。

## 257 二叉树的所有路径

[题目链接](https://leetcode.cn/problems/binary-tree-paths/)

### 257 题意

给定一棵二叉树的根节点 `root`，
返回所有从根节点到叶子节点的路径。

叶子节点是没有左孩子也没有右孩子的节点。

## 方法一：迭代 DFS

### 迭代思路

迭代 DFS 需要用栈模拟递归。

递归调用栈天然能记录当前节点以及路径。
改成迭代后，需要自己把节点和路径绑定在一起。

因此定义 `PathState`：

- `Node` 表示当前节点。
- `Path` 表示从根节点走到当前节点的路径字符串。

当遇到叶子节点时，当前 `Path` 就是一条完整路径。

### 迭代代码

```go
type PathState struct {
    Node *TreeNode
    Path string
}

func binaryTreePaths(root *TreeNode) []string {
    if root == nil {
        return []string{}
    }

    ans := []string{}
    stack := []PathState{}

    stack = append(stack, PathState{
        Node: root,
        Path: strconv.Itoa(root.Val),
    })

    for len(stack) > 0 {
        state := stack[len(stack)-1]
        stack = stack[:len(stack)-1]

        node := state.Node
        path := state.Path

        if node.Left == nil && node.Right == nil {
            ans = append(ans, path)
        }

        if node.Right != nil {
            stack = append(stack, PathState{
                Node: node.Right,
                Path: path + "->" + strconv.Itoa(node.Right.Val),
            })
        }

        if node.Left != nil {
            stack = append(stack, PathState{
                Node: node.Left,
                Path: path + "->" + strconv.Itoa(node.Left.Val),
            })
        }
    }

    return ans
}
```

### 迭代顺序说明

代码先压入右孩子，再压入左孩子。

因为栈是后进先出，所以左孩子会先被处理。
这样实际遍历顺序更接近前序 DFS。

## 方法二：递归回溯

### 回溯思路

递归写法可以用 `path` 保存当前路径上的节点值。

处理过程：

1. 访问当前节点，把节点值加入 `path`。
2. 如果当前节点是叶子节点，把 `path` 转成字符串加入结果。
3. 递归访问左子树。
4. 递归访问右子树。
5. 回退时弹出当前节点。

最后一步 `path = path[:len(path)-1]` 是回溯的关键。

### 回溯代码

```go
func binaryTreePaths(root *TreeNode) []string {
    var res []string
    var path []int

    var backtracking func(*TreeNode)
    backtracking = func(node *TreeNode) {
        if node == nil {
            return
        }

        path = append(path, node.Val)

        if node.Left == nil && node.Right == nil {
            var pathStr []string
            for _, v := range path {
                pathStr = append(pathStr, strconv.Itoa(v))
            }
            res = append(res, strings.Join(pathStr, "->"))
        }

        backtracking(node.Left)
        backtracking(node.Right)

        path = path[:len(path)-1]
    }

    backtracking(root)
    return res
}
```

## 复杂度

- 时间复杂度：`O(n * h)`
- 空间复杂度：`O(h)`

其中 `n` 是节点数量，`h` 是树高。

路径转换成字符串时，需要遍历当前路径，
因此总时间和路径长度有关。

## 本题总结

这道题的核心是 DFS 过程中保存当前路径。

迭代法需要把节点和路径一起放入栈中。
递归法则可以通过函数调用栈和回溯维护路径。

无论哪种写法，遇到叶子节点时，当前路径就是完整答案。
