# 二叉搜索树小结：中序有序与查找导航

二叉搜索树有两个非常重要的性质。

第一，BST 的中序遍历结果是有序数组。

第二，BST 具有查找导航能力：
每一步都能根据当前节点值决定向左还是向右。

## 中序遍历有序

很多 BST 题目可以通过中序遍历转化成有序数组问题。

### 98 验证二叉搜索树

[题目链接](https://leetcode.cn/problems/validate-binary-search-tree/)

树的问题：

```text
判断一棵树是不是有效 BST
```

数组视角：

```text
判断中序遍历结果是否严格递增
```

### 530 二叉搜索树的最小绝对差

[题目链接](https://leetcode.cn/problems/minimum-absolute-difference-in-bst/)

树的问题：

```text
找树中任意两个节点值之间的最小差值
```

数组视角：

```text
在有序数组中找相邻元素的最小差值
```

### 501 二叉搜索树中的众数

[题目链接](https://leetcode.cn/problems/find-mode-in-binary-search-tree/)

树的问题：

```text
找树中出现频率最高的元素
```

数组视角：

```text
在有序数组中统计连续相同元素的最长段
```

因为中序遍历有序，相同元素会连续出现。
这让遍历过程中的连续计数变得可行。

## 查找导航能力

BST 还有另一个重要能力：根据当前节点值决定搜索方向。

例如：

[700 二叉搜索树中的搜索](https://leetcode.cn/problems/search-in-a-binary-search-tree/)

如果目标值小于当前节点值，只需要去左子树。

如果目标值大于当前节点值，只需要去右子树。

这类问题利用的是 BST 的二分式导航能力。

未来的插入、删除等操作，也会大量依赖这个性质。

## 本篇总结

BST 题目可以先判断它主要利用哪种性质。

如果题目和节点值顺序、大小关系、相邻关系有关，
优先考虑中序遍历。

如果题目和查找、插入、删除某个目标值有关，
优先考虑 BST 的导航性质。

这两个性质是理解 BST 题目的核心。
