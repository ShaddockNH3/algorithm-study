# 建造最大岛屿

[卡码网 104. 建造最大岛屿](https://kamacoder.com/problempage.php?pid=1176)

## 核心思路

这题不能暴力尝试把每个 `0` 改成 `1` 后再重新 DFS。

更高效的做法是两步走。

第一步：
遍历整张图，
用 DFS 给每个已有岛屿标记唯一编号，
同时记录每个编号对应的岛屿面积。

第二步：
遍历每个水格子 `0`，
查看它上下左右相邻的岛屿编号。
把不同编号的岛屿面积加起来，
再加上当前水格子变成陆地后的面积 `1`。

为了避免同一个岛屿从多个方向被重复计算，
需要用一个 `map[int]bool` 去重。

## 错误复盘

原文代码里有几处需要修正：

- 访问方向数组时要写 `dir[k][0]` 和 `dir[k][1]`。
- `map` 不能写成 `mp = 0`，应使用 `mp[0] = 0`。
- DFS 统计面积时，不能遇到一个方向就直接 `return`。
- 计算水格子相邻岛屿时，要用集合避免重复加同一个岛屿。

## 修正版代码

```go
package main

import "fmt"

var dir = [][]int{
    {0, 1}, {1, 0}, {-1, 0}, {0, -1},
}

func solve() {
    var n, m int
    fmt.Scanln(&n, &m)

    graph := make([][]int, n)
    visit := make([][]bool, n)

    for i := 0; i < n; i++ {
        graph[i] = make([]int, m)
        visit[i] = make([]bool, m)
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            fmt.Scan(&graph[i][j])
        }
    }

    var dfs func(x, y, id int) int
    dfs = func(x, y, id int) int {
        visit[x][y] = true
        graph[x][y] = id

        area := 1
        for k := 0; k < 4; k++ {
            nextX := x + dir[k][0]
            nextY := y + dir[k][1]

            if nextX < 0 || nextX >= n || nextY < 0 || nextY >= m {
                continue
            }

            if !visit[nextX][nextY] && graph[nextX][nextY] == 1 {
                area += dfs(nextX, nextY, id)
            }
        }

        return area
    }

    islandArea := make(map[int]int)
    islandArea[0] = 0

    id := 2
    maxArea := 0
    hasZero := false

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if !visit[i][j] && graph[i][j] == 1 {
                area := dfs(i, j, id)
                islandArea[id] = area
                maxArea = max(maxArea, area)
                id++
            }
        }
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if graph[i][j] != 0 {
                continue
            }

            hasZero = true
            current := 1
            seen := make(map[int]bool)

            for k := 0; k < 4; k++ {
                nextX := i + dir[k][0]
                nextY := j + dir[k][1]

                if nextX < 0 || nextX >= n || nextY < 0 || nextY >= m {
                    continue
                }

                islandID := graph[nextX][nextY]
                if islandID > 1 && !seen[islandID] {
                    current += islandArea[islandID]
                    seen[islandID] = true
                }
            }

            maxArea = max(maxArea, current)
        }
    }

    if !hasZero {
        fmt.Println(n * m)
        return
    }

    fmt.Println(maxArea)
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}

func main() {
    solve()
}
```

## 本篇总结

建造最大岛屿的关键是先给已有岛屿编号并记录面积。

之后每个水格子只需要查看四周不同岛屿编号，
就能快速计算把它变成陆地后的总面积。

去重集合非常重要，
否则同一个岛屿可能从多个方向被重复累加。
