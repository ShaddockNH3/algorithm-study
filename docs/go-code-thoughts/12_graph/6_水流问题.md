# 水流问题

[卡码网 103. 水流问题](https://kamacoder.com/problempage.php?pid=1175)

## 核心思路

不要从每个格子出发模拟水往低处流。

可以反过来，
从边界出发往高处搜索。

如果水能从某个格子流到边界，
那么反向看，
就可以从边界沿着高度不降低的方向走到这个格子。

准备两个访问数组：

- `visit1`：能到达第一组边界的格子。
- `visit2`：能到达第二组边界的格子。

分别从两组边界出发做 DFS。

最后同时满足 `visit1[i][j] && visit2[i][j]` 的格子就是答案。

## 错误复盘

原来遇到的问题是 DFS 没有及时判断是否访问过。

如果两个相邻格子高度允许互相走，
DFS 可能从 A 到 B，
又从 B 回到 A，
导致重复递归甚至栈溢出。

因此每个访问数组都必须阻止重复访问。

## 原始区分编号写法

```go
package main

import "fmt"

var dir = [][]int{
    {0, 1}, {1, 0}, {-1, 0}, {0, -1},
}

func solve() {
    var n, m int
    fmt.Scanln(&n, &m)

    grid := make([][]int, n)
    visit1 := make([][]bool, n)
    visit2 := make([][]bool, n)

    for i := range grid {
        grid[i] = make([]int, m)
        visit1[i] = make([]bool, m)
        visit2[i] = make([]bool, m)
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            fmt.Scan(&grid[i][j])
        }
    }

    var dfs func(x, y, area int)
    dfs = func(x, y, area int) {
        if area == 1 {
            visit1[x][y] = true
        } else {
            visit2[x][y] = true
        }

        for i := 0; i < 4; i++ {
            nextX := x + dir[i][0]
            nextY := y + dir[i][1]

            if nextX < 0 || nextX >= n || nextY < 0 || nextY >= m {
                continue
            }

            if area == 1 && visit1[nextX][nextY] {
                continue
            }
            if area == 2 && visit2[nextX][nextY] {
                continue
            }

            if grid[nextX][nextY] >= grid[x][y] {
                dfs(nextX, nextY, area)
            }
        }
    }

    for i := 0; i < n; i++ {
        dfs(i, 0, 1)
    }
    for j := 0; j < m; j++ {
        dfs(0, j, 1)
    }

    for i := 0; i < n; i++ {
        dfs(i, m-1, 2)
    }
    for j := 0; j < m; j++ {
        dfs(n-1, j, 2)
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if visit1[i][j] && visit2[i][j] {
                fmt.Println(i, j)
            }
        }
    }
}
```

## 更通用的 dfs 写法

可以把 `visit` 作为参数传给 DFS，
避免用 `area` 区分访问数组。

```go
var dfs func(x, y int, visit [][]bool)
dfs = func(x, y int, visit [][]bool) {
    if visit[x][y] {
        return
    }

    visit[x][y] = true

    for i := 0; i < 4; i++ {
        nextX := x + dir[i][0]
        nextY := y + dir[i][1]

        if nextX < 0 || nextX >= n || nextY < 0 || nextY >= m {
            continue
        }

        if grid[nextX][nextY] >= grid[x][y] {
            dfs(nextX, nextY, visit)
        }
    }
}
```

调用时直接传入对应访问数组：

```go
dfs(i, 0, visit1)
dfs(0, j, visit1)
dfs(i, m-1, visit2)
dfs(n-1, j, visit2)
```

## 本篇总结

水流问题适合反向搜索。

从边界往高度更高或相等的位置走，
分别得到能到达两组边界的格子集合。

两个集合的交集就是答案。
