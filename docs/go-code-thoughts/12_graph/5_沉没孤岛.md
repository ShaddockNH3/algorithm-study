# 沉没孤岛

[卡码网 102. 沉没孤岛](https://kamacoder.com/problempage.php?pid=1174)

## 核心思路

题目要把不连接边界的孤岛沉没。

直接找孤岛会比较麻烦，
可以反过来找不会沉没的陆地。

不会沉没的陆地包括：

- 边界上的陆地。
- 和边界陆地连通的陆地。

做法是：

1. 从四条边界出发，找到所有 `1`。
2. 对这些边界陆地做 DFS，把连通陆地标记为 `2`。
3. 再遍历整张图。
4. 仍然是 `1` 的陆地沉没为 `0`。
5. 标记为 `2` 的安全陆地恢复成 `1`。

## 注意点

原笔记里提到一个打印问题：

打印结果时应该打印 `graph[i][j]` 的值，
不要打印 `&graph[i][j]` 的地址。

## 最终代码

```go
package main

import "fmt"

var dir = [][]int{
    {0, 1}, {1, 0}, {-1, 0}, {0, -1},
}

func solve() {
    var n, m int
    fmt.Scanln(&n, &m)

    graph := make([][]int, n)
    for i := range graph {
        graph[i] = make([]int, m)
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            fmt.Scan(&graph[i][j])
        }
    }

    var dfs func(x, y int)
    dfs = func(x, y int) {
        graph[x][y] = 2

        for i := 0; i < 4; i++ {
            nextX := x + dir[i][0]
            nextY := y + dir[i][1]

            if nextX < 0 || nextX >= n || nextY < 0 || nextY >= m {
                continue
            }

            if graph[nextX][nextY] == 1 {
                dfs(nextX, nextY)
            }
        }
    }

    for i := 0; i < n; i++ {
        if graph[i][0] == 1 {
            dfs(i, 0)
        }
        if graph[i][m-1] == 1 {
            dfs(i, m-1)
        }
    }

    for j := 0; j < m; j++ {
        if graph[0][j] == 1 {
            dfs(0, j)
        }
        if graph[n-1][j] == 1 {
            dfs(n-1, j)
        }
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if graph[i][j] == 1 {
                graph[i][j] = 0
            } else if graph[i][j] == 2 {
                graph[i][j] = 1
            }
        }

        for j := 0; j < m; j++ {
            fmt.Printf("%d ", graph[i][j])
        }
        fmt.Println()
    }
}

func main() {
    solve()
}
```

## 本篇总结

沉没孤岛的关键是逆向思维。

先标记所有和边界连通的安全陆地，
再把剩下的陆地沉没。

这样不用单独判断某个岛屿是否孤立。
