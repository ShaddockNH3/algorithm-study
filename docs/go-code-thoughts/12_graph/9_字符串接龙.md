## 核心思路

这是无权图最短路径问题，
适合使用 BFS。

可以把每个单词看成一个节点。

如果两个单词只差一个字母，
它们之间就有一条边。

目标是从 `beginStr` 到 `endStr` 的最短转换序列长度。

BFS 一层一层扩展，
第一次遇到终点时，
对应的路径长度就是最短长度。

## 生成邻居

朴素做法是拿当前单词和字典里所有单词比较。

更高效的做法是生成邻居：

对当前单词的每一个位置，
尝试替换成 `'a'` 到 `'z'`，
生成一个新单词。

如果新单词在字典中，
并且没有访问过，
就加入 BFS 队列。

## 最终代码

```go
package main

import "fmt"

func solve() {
    var n int
    fmt.Scanln(&n)

    var beginStr, endStr string
    fmt.Scanln(&beginStr, &endStr)

    strSet := make(map[string]struct{})
    for i := 0; i < n; i++ {
        var s string
        fmt.Scanln(&s)
        strSet[s] = struct{}{}
    }

    visitMap := make(map[string]int)
    queue := []string{beginStr}
    visitMap[beginStr] = 1

    for len(queue) > 0 {
        word := queue[0]
        queue = queue[1:]

        path := visitMap[word]
        newWordRunes := []rune(word)

        for i := 0; i < len(newWordRunes); i++ {
            originalChar := newWordRunes[i]

            for ch := 'a'; ch <= 'z'; ch++ {
                if ch == originalChar {
                    continue
                }

                newWordRunes[i] = ch
                newWord := string(newWordRunes)

                if newWord == endStr {
                    fmt.Println(path + 1)
                    return
                }

                if _, inSet := strSet[newWord]; inSet {
                    if _, visited := visitMap[newWord]; !visited {
                        visitMap[newWord] = path + 1
                        queue = append(queue, newWord)
                    }
                }
            }

            newWordRunes[i] = originalChar
        }
    }

    fmt.Println(0)
}

func main() {
    solve()
}
```

## 本篇总结

字符串接龙是典型 BFS 最短路。

`visitMap` 同时承担两个作用：

- 防止重复访问。
- 记录到达当前单词的步数。

生成邻居时，
逐位替换字母再查字典，
比遍历整个字典逐个比较更直接。
