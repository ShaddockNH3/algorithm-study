# 深度优先搜索模板

深度优先搜索可以理解为：
沿着一条路径不断深入，
直到无法继续或达到目标，
再回退到上一个节点尝试其他分支。

## 通用模板

```cpp
void dfs(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本节点所连接的其他节点) {
        处理节点;
        dfs(图，选择的节点);
        回溯，撤销处理结果;
    }
}
```

## 本篇总结

DFS 的核心是递归深入。

如果题目要求记录路径或枚举所有方案，
就常常需要在递归返回后撤销选择，
这就是回溯。
