# 岛屿的周长

[卡码网 106. 岛屿的周长](https://kamacoder.com/problempage.php?pid=1178)

## 核心思路

这题不需要 DFS 或 BFS。

周长来自陆地和非陆地的交界。

遍历每个格子，
只处理值为 `1` 的陆地。

对每块陆地检查四个方向：

- 如果邻居越界，贡献一条边。
- 如果邻居是水 `0`，贡献一条边。
- 如果邻居也是陆地，不贡献周长。

把所有陆地的贡献累加起来，
就是总周长。

## 笔误复盘

原来出现过一个坐标笔误：

```go
y := i + dir[k][1]
```

这里计算列坐标时，
应该使用当前列 `j`，
而不是当前行 `i`。

正确写法是：

```go
y := j + dir[k][1]
```

## 最终代码

```go
package main

import "fmt"

var dir = [][]int{
    {0, 1}, {1, 0}, {-1, 0}, {0, -1},
}

func solve() {
    var n, m int
    fmt.Scanln(&n, &m)

    graph := make([][]int, n)
    for i := 0; i < n; i++ {
        graph[i] = make([]int, m)
    }

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            fmt.Scan(&graph[i][j])
        }
    }

    result := 0

    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if graph[i][j] != 1 {
                continue
            }

            for k := 0; k < 4; k++ {
                x := i + dir[k][0]
                y := j + dir[k][1]

                if x < 0 || x >= n || y < 0 || y >= m || graph[x][y] == 0 {
                    result++
                }
            }
        }
    }

    fmt.Println(result)
}

func main() {
    solve()
}
```

## 本篇总结

岛屿周长可以用局部贡献法解决。

每块陆地最多贡献四条边。
只要某个方向是水或边界，
这个方向就贡献一条周长。

不需要判断连通块，
也不需要搜索整座岛。
