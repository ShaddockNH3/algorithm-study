# 17 电话号码的字母组合

[题目链接](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)

## 题目描述

给定一个仅包含数字 `2-9` 的字符串，
返回所有它能表示的字母组合。

答案可以按任意顺序返回。

数字到字母的映射和电话按键相同，
其中 `1` 不对应任何字母。

## 识别回溯模型

当题目要求返回所有可能组合时，
可以考虑回溯。

例如输入为 `23` 时：

- 第一层处理数字 `2`，可选字母是 `a`、`b`、`c`。
- 第二层处理数字 `3`，可选字母是 `d`、`e`、`f`。
- 当路径长度等于输入数字长度时，得到一个完整组合。

这就是一棵决策树。
每一层对应输入字符串中的一个数字，
这一层的所有分支就是这个数字对应的字母。

## 回溯函数设计

回溯函数需要知道以下信息：

- `digits`：原始数字字符串。
- `start`：当前处理到第几个数字。
- `path`：当前已经拼出的字母路径。
- `res`：保存所有结果。
- `mp`：数字到字母的映射表。

这里使用 `[]byte` 记录路径，
比频繁拼接字符串更方便做追加和回退。

## 终止条件

当 `len(path) == len(digits)` 时，
说明已经处理完所有数字，
当前路径就是一个完整组合。

此时要复制 `path`，
再转成字符串放入结果集。

复制的原因是 `path` 后续还会被继续回溯复用。

## 核心循环

在每一层中：

1. 根据 `digits[start]` 找到当前数字。
2. 从 `mp` 中取出这个数字对应的字母字符串。
3. 遍历这些字母。
4. 选择一个字母，进入下一层。
5. 返回后撤销选择，尝试下一个字母。

这道题的目标是遍历所有可能组合，
不存在明显的无效分支，
所以不需要剪枝。

## 最终代码

```go
func letterCombinations(digits string) []string {
    if len(digits) == 0 {
        return []string{}
    }

    res := []string{}
    path := []byte{}
    mp := map[byte]string{
        '2': "abc",
        '3': "def",
        '4': "ghi",
        '5': "jkl",
        '6': "mno",
        '7': "pqrs",
        '8': "tuv",
        '9': "wxyz",
    }

    backtracking(digits, &res, &path, mp, 0)
    return res
}

func backtracking(
    digits string,
    res *[]string,
    path *[]byte,
    mp map[byte]string,
    start int,
) {
    if len(*path) == len(digits) {
        tmp := make([]byte, len(*path))
        copy(tmp, *path)
        *res = append(*res, string(tmp))
        return
    }

    letters := mp[digits[start]]

    for i := 0; i < len(letters); i++ {
        letter := letters[i]

        *path = append(*path, letter)
        backtracking(digits, res, path, mp, start+1)
        *path = (*path)[:len(*path)-1]
    }
}
```

## 本篇总结

电话号码组合的每一层对应一个数字，
每层的选择集合是这个数字映射出的字母。

这道题重点是理解决策树层级：
`start` 控制当前处理哪个数字，
`path` 保存当前路径。

因为要枚举所有组合，所以这里不需要剪枝。
