# 131 分割回文串

[题目链接](https://leetcode.cn/problems/palindrome-partitioning/)

## 题目描述

给你一个字符串 `s`，
请你将 `s` 分割成一些子串，
使每个子串都是回文串。

返回 `s` 所有可能的分割方案。

## 问题本质

这道题是切割问题。

要求所有可能的切割方案，
并且每一段都必须是回文串。
这类求所有方案的问题很适合用回溯。

可以把字符串想象成一段需要被切开的区间。
每一层递归都决定下一刀切在哪里。

## 回溯函数的含义

`backtracking(startIndex)` 的含义是：

从字符串 `s` 的 `startIndex` 位置开始，
寻找后续所有合法的回文分割方案。

`startIndex` 控制当前从哪里开始切。
`path` 保存已经切出来的回文子串。

## 如何枚举切割点

站在 `startIndex` 位置时，
可以尝试把当前这一段切成：

- `s[startIndex:startIndex+1]`
- `s[startIndex:startIndex+2]`
- 一直到 `s[startIndex:len(s)]`

所以使用循环：

```go
for i := startIndex; i < len(s); i++ {
}
```

这里的 `i` 表示当前尝试切割的子串右边界。
实际子串是：

```go
s[startIndex : i+1]
```

## 合法性判断

每次切出一个子串后，
都要判断它是不是回文串。

只有当 `isPalindrome(s, startIndex, i)` 为真时，
才能把这段加入 `path`，
然后递归处理 `i + 1` 后面的剩余字符串。

如果不是回文串，
就继续扩大切割范围，尝试下一种切法。

## 终止条件

当 `startIndex == len(s)` 时，
说明整个字符串已经被切割完。

此时 `path` 中保存的是一组完整的合法分割方案，
需要复制一份加入结果集。

复制的原因是 `path` 之后还会继续被回溯复用。

## 最终代码

```go
func partition(s string) [][]string {
    res := [][]string{}
    path := []string{}

    backtracking(s, &res, &path, 0)
    return res
}

func backtracking(s string, res *[][]string, path *[]string, startIndex int) {
    if startIndex == len(s) {
        tmp := make([]string, len(*path))
        copy(tmp, *path)
        *res = append(*res, tmp)
        return
    }

    for i := startIndex; i < len(s); i++ {
        if isPalindrome(s, startIndex, i) {
            str := s[startIndex : i+1]

            *path = append(*path, str)
            backtracking(s, res, path, i+1)
            *path = (*path)[:len(*path)-1]
        } else {
            continue
        }
    }
}

func isPalindrome(s string, left int, right int) bool {
    for left < right {
        if s[left] != s[right] {
            return false
        }
        left++
        right--
    }
    return true
}
```

## 字符处理补充

这里的 `isPalindrome` 直接按字节比较，
对于纯英文字符串或力扣本题输入是合适的。

如果要更完整地支持中文等多字节字符，
可以先把字符串转成 `[]rune` 再判断。

## 本篇总结

分割回文串的关键是把问题看成切割决策树。

`for` 循环负责枚举当前这一刀的结束位置，
递归负责继续处理剩余字符串，
回溯负责撤销当前切割，尝试下一种切法。

终止条件是整个字符串被切完，
而不是简单地达到某个固定层数。
