# 回溯模板

回溯法可以抽象成一棵树。

在这棵树里：

- `for` 循环负责横向遍历本层可以选择的元素。
- 递归负责纵向进入下一层。
- 回溯负责撤销当前选择，继续尝试本层的下一个元素。

## 通用模板

```cpp
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表);
        回溯，撤销处理结果;
    }
}
```

## 理解方式

回溯法抽象成树形结构之后，
遍历过程就是：

```text
for 循环横向遍历
递归纵向遍历
回溯不断调整结果集
```

`for` 循环的长度可以理解为树的宽度，
递归的深度可以理解为树的高度。
