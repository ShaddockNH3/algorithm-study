## 209 长度最小的子数组

[题目链接](https://leetcode.cn/problems/minimum-size-subarray-sum/)

### 209 题意

给定一个含有 `n` 个正整数的数组和一个正整数 `target`。

找出该数组中满足总和大于等于 `target` 的长度最小的子数组，
并返回其长度。如果不存在符合条件的子数组，返回 `0`。

### 209 示例

示例 1：

```text
输入：target = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。
```

示例 2：

```text
输入：target = 4, nums = [1,4,4]
输出：1
```

示例 3：

```text
输入：target = 11, nums = [1,1,1,1,1,1,1,1]
输出：0
```

### 209 提示

- `1 <= target <= 10^9`
- `1 <= nums.length <= 10^5`
- `1 <= nums[i] <= 10^4`

进阶要求：

- 已经实现 `O(n)` 解法后，可以尝试设计 `O(n log n)` 解法。

### 209 思路

因为数组元素都是正整数，窗口右端向右扩张时，窗口和会增大；
窗口左端向右收缩时，窗口和会减小。

这满足滑动窗口的使用条件。

具体步骤：

1. 使用 `right` 扩张窗口，并把 `nums[right]` 加入当前和。
2. 当 `currentSum >= target` 时，说明当前窗口满足条件。
3. 尝试更新最小长度。
4. 不断移动 `left` 收缩窗口，寻找更短答案。
5. 如果最后没有更新过答案，返回 `0`。

### 209 代码

```go
func minSubArrayLen(target int, nums []int) int {
    n := len(nums)
    left, right := 0, 0
    minLength := n + 1
    currentSum := 0

    for ; right < n; right++ {
        currentSum += nums[right]
        for currentSum >= target && left <= right {
            minLength = min(minLength, right-left+1)
            currentSum -= nums[left]
            left++
        }
    }

    if minLength == n+1 {
        return 0
    }
    return minLength
}
```

### 209 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

`left` 和 `right` 都只会向右移动，整体不会重复遍历。

## 904 水果成篮

[题目链接](https://leetcode.cn/problems/fruit-into-baskets/)

### 904 题意

农场从左到右种植了一排果树。数组 `fruits` 表示果树，
其中 `fruits[i]` 是第 `i` 棵树上的水果种类。

采摘规则如下：

- 只有两个篮子。
- 每个篮子只能装一种类型的水果。
- 每个篮子的容量没有限制。
- 可以选择任意一棵树开始采摘。
- 之后必须从每棵树上恰好摘一个水果，并一直向右移动。
- 一旦遇到不符合篮子类型的水果，就必须停止采摘。

返回可以收集的水果最大数量。

### 904 示例

示例 1：

```text
输入：fruits = [1,2,1]
输出：3
解释：可以采摘全部 3 棵树。
```

示例 2：

```text
输入：fruits = [0,1,2,2]
输出：3
解释：可以采摘 [1,2,2] 这三棵树。
如果从第一棵树开始采摘，则只能采摘 [0,1] 这两棵树。
```

示例 3：

```text
输入：fruits = [1,2,3,2,2]
输出：4
解释：可以采摘 [2,3,2,2] 这四棵树。
如果从第一棵树开始采摘，则只能采摘 [1,2] 这两棵树。
```

示例 4：

```text
输入：fruits = [3,3,3,1,2,1,1,2,3,3,4]
输出：5
解释：可以采摘 [1,2,1,1,2] 这五棵树。
```

### 904 提示

- `1 <= fruits.length <= 10^5`
- `0 <= fruits[i] < fruits.length`

## 本篇总结

209 和 904 都可以用滑动窗口理解。

- 209 的窗口约束是窗口和至少为 `target`。
- 904 的窗口约束是窗口内最多包含两种水果。

滑动窗口的重点是明确窗口什么时候扩张，什么时候收缩，
以及窗口内需要维护哪些状态。
