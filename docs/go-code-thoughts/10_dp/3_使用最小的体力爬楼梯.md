# 746 使用最小花费爬楼梯

[题目链接](https://leetcode.cn/problems/min-cost-climbing-stairs/)

## 题目描述

给你一个整数数组 `cost`，
其中 `cost[i]` 是从第 `i` 个台阶向上爬需要支付的费用。

支付费用后，
可以选择向上爬 1 个或 2 个台阶。

可以从下标 `0` 或下标 `1` 的台阶开始爬。

计算到达楼梯顶部的最低花费。

## 理解题意

如果 `cost` 的下标是 `0` 到 `len(cost)-1`，
那么楼梯顶部可以理解为下标 `len(cost)` 的位置。

踏上起点本身不花钱。
只有从某个台阶继续向上爬时，
才支付这个台阶对应的 `cost`。

## dp 数组定义

定义：

`dp[i]` 表示到达第 `i` 阶楼梯时的累计最小费用。

最终要求的是：

```go
dp[len(cost)]
```

因此需要创建长度为 `len(cost) + 1` 的 `dp` 数组。

## 状态转移

要到达第 `i` 阶，
可能来自两个位置。

从第 `i-1` 阶爬 1 步上来：

```text
dp[i-1] + cost[i-1]
```

从第 `i-2` 阶爬 2 步上来：

```text
dp[i-2] + cost[i-2]
```

取两者较小值：

```text
dp[i] = min(dp[i-1] + cost[i-1], dp[i-2] + cost[i-2])
```

## 初始化

可以从第 `0` 阶或第 `1` 阶开始，
并且开始时不需要付费。

所以：

```text
dp[0] = 0
dp[1] = 0
```

## 最终代码

```go
func minCostClimbingStairs(cost []int) int {
    dp := make([]int, len(cost)+1)
    dp[0], dp[1] = 0, 0

    for i := 2; i <= len(cost); i++ {
        dp[i] = min(dp[i-2]+cost[i-2], dp[i-1]+cost[i-1])
    }

    return dp[len(cost)]
}

func min(a, b int) int {
    if a < b {
        return a
    }
    return b
}
```

## 示例推导

以 `cost = [10, 15, 20]` 为例。

初始：

```text
dp = [0, 0, ?, ?]
```

当 `i = 2`：

```text
dp[2] = min(dp[0] + cost[0], dp[1] + cost[1])
      = min(0 + 10, 0 + 15)
      = 10
```

当 `i = 3`：

```text
dp[3] = min(dp[1] + cost[1], dp[2] + cost[2])
      = min(0 + 15, 10 + 20)
      = 15
```

返回 `dp[3] = 15`。

这表示从第 1 阶开始，
支付 `cost[1] = 15`，
直接爬两步到达顶部。

## 本篇总结

这道题是爬楼梯模型的变体。

普通爬楼梯求方法数，
这道题求最小花费。

关键是把 `dp[i]` 定义成到达第 `i` 阶的最小累计费用，
并注意顶部是 `len(cost)` 这个位置。
