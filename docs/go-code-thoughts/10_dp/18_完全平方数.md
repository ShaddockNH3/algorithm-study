# 279 完全平方数

[题目链接](https://leetcode.cn/problems/perfect-squares/)

## 题目描述

给你一个整数 `n`，
返回和为 `n` 的完全平方数的最少数量。

每个完全平方数可以重复使用。

## 问题模型

这道题可以看成零钱兑换问题的变形。

对应关系是：

- 零钱兑换中的总金额，对应这里的目标整数 `n`。
- 零钱兑换中的硬币列表，对应完全平方数列表 `1, 4, 9, ...`。
- 硬币可以无限使用，对应完全平方数可以无限使用。
- 求最少硬币数量，对应求最少完全平方数数量。

所以这也是完全背包的最小值问题。

## dp 数组定义

定义：

`dp[j]` 表示和为 `j` 的完全平方数的最少数量。

初始化：

- `dp[0] = 0`
- 其他位置初始化为 `n + 1`，表示暂时不可达

状态转移：

```text
dp[j] = min(dp[j], dp[j-num] + 1)
```

其中 `num` 是某个完全平方数。

## 生成完全平方数列表

先生成所有小于等于 `n` 的完全平方数。

```go
func getNums(n int) []int {
    nums := []int{}
    for num := 1; num*num <= n; num++ {
        nums = append(nums, num*num)
    }
    return nums
}
```

## 最终代码

```go
func numSquares(n int) int {
    nums := getNums(n)
    dp := make([]int, n+1)

    for i := 1; i <= n; i++ {
        dp[i] = n + 1
    }

    for _, num := range nums {
        for j := num; j <= n; j++ {
            dp[j] = min(dp[j], dp[j-num]+1)
        }
    }

    return dp[n]
}

func getNums(n int) []int {
    nums := []int{}
    for num := 1; num*num <= n; num++ {
        nums = append(nums, num*num)
    }
    return nums
}

func min(a, b int) int {
    if a < b {
        return a
    }
    return b
}
```

## 本篇总结

完全平方数和零钱兑换是同一类模型。

只要把硬币列表换成完全平方数列表，
就可以复用完全背包最小值问题的写法：

```text
dp[j] = min(dp[j], dp[j-num] + 1)
```
