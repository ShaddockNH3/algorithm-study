# 714 买卖股票的最佳时机含手续费

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-transaction-fee/)

## 题目描述

给定股票价格数组 `prices` 和手续费 `fee`。

可以进行多次交易，
但每次卖出股票时需要支付一笔手续费。

同一时间最多只能持有一股股票。

返回最大利润。

## 状态定义

每天结束时有两种状态：

- `dp[i][0]`：第 `i` 天结束时持有股票的最大利润。
- `dp[i][1]`：第 `i` 天结束时不持有股票的最大利润。

## 状态转移

第 `i` 天持有股票：

```text
dp[i][0] = max(dp[i-1][0], dp[i-1][1] - prices[i])
```

含义是：

- 昨天就持有，今天继续持有。
- 昨天不持有，今天买入。

第 `i` 天不持有股票：

```text
dp[i][1] = max(dp[i-1][1], dp[i-1][0] + prices[i] - fee)
```

含义是：

- 昨天就不持有，今天继续不持有。
- 昨天持有，今天卖出，并扣除手续费。

## 初始化

第 0 天：

```text
dp[0][0] = -prices[0]
dp[0][1] = 0
```

## 二维 DP 代码

```go
func maxProfitDP(prices []int, fee int) int {
    if len(prices) == 0 {
        return 0
    }

    dp := make([][]int, len(prices))
    for i := range dp {
        dp[i] = make([]int, 2)
    }

    dp[0][0] = -prices[0]

    for i := 1; i < len(prices); i++ {
        dp[i][0] = max(dp[i-1][0], dp[i-1][1]-prices[i])
        dp[i][1] = max(dp[i-1][1], prices[i]+dp[i-1][0]-fee)
    }

    return dp[len(prices)-1][1]
}
```

## 空间优化

只依赖前一天状态，
所以可以用两个变量：

- `hold`：持有股票状态。
- `cash`：不持有股票状态。

更新时先保存旧的 `cash`，
避免今天状态互相污染。

```go
func maxProfit(prices []int, fee int) int {
    if len(prices) == 0 {
        return 0
    }

    hold := -prices[0]
    cash := 0

    for i := 1; i < len(prices); i++ {
        oldCash := cash
        cash = max(cash, hold+prices[i]-fee)
        hold = max(hold, oldCash-prices[i])
    }

    return cash
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

含手续费股票问题和无限次交易很像。

区别只在卖出时扣除手续费：

```text
dp[i-1][0] + prices[i] - fee
```

状态仍然是持有和不持有两类，
可以很自然地做空间压缩。
