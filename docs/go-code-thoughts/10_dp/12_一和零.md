# 474 一和零

[题目链接](https://leetcode.cn/problems/ones-and-zeroes/)

## 题目描述

给你一个二进制字符串数组 `strs`，
以及两个整数 `m` 和 `n`。

请找出并返回 `strs` 的最大子集长度，
该子集中最多有 `m` 个 `0` 和 `n` 个 `1`。

每个字符串最多只能使用一次。

## 问题转化

字符串本身不是重点。

真正重要的是每个字符串中 `0` 和 `1` 的数量。

例如：

```text
"10"   -> 1 个 0, 1 个 1
"0001" -> 3 个 0, 1 个 1
"1"    -> 0 个 0, 1 个 1
```

这道题可以看成二维费用的 01 背包：

- 每个字符串是一个物品。
- 这个物品有两种重量：`0` 的数量和 `1` 的数量。
- 背包有两种容量：最多 `m` 个 `0`，最多 `n` 个 `1`。
- 每个物品的价值都是 `1`，因为目标是最大子集长度。

## dp 数组定义

定义：

`dp[i][j]` 表示最多使用 `i` 个 `0`
和 `j` 个 `1` 时，
能够得到的最大子集长度。

最终要求：

```go
dp[m][n]
```

## 状态转移

对每个字符串，
先统计它消耗的 `0` 和 `1` 数量：

```text
num0, num1
```

然后做 01 背包决策。

不放当前字符串：

```text
dp[i][j]
```

放当前字符串：

```text
dp[i-num0][j-num1] + 1
```

取最大值：

```text
dp[i][j] = max(dp[i][j], dp[i-num0][j-num1] + 1)
```

## 为什么倒序遍历

如果没有空间优化，
可以用三维数组：

```text
dp[k][i][j]
```

表示只考虑前 `k` 个字符串时的结果。

现在去掉了物品维度，
直接在二维数组上滚动更新。

为了保证每个字符串只使用一次，
两个容量维度都必须倒序遍历：

```go
for i := m; i >= num0; i-- {
    for j := n; j >= num1; j-- {
    }
}
```

如果正序遍历，
`dp[i-num0][j-num1]` 可能已经在本轮被当前字符串更新过，
这就相当于同一个字符串被重复使用。

## 最终代码

```go
func findMaxForm(strs []string, m int, n int) int {
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }

    for _, str := range strs {
        num0, num1 := getNum(str)

        for i := m; i >= num0; i-- {
            for j := n; j >= num1; j-- {
                dp[i][j] = max(dp[i][j], dp[i-num0][j-num1]+1)
            }
        }
    }

    return dp[m][n]
}

func getNum(str string) (int, int) {
    num0, num1 := 0, 0
    for _, ch := range str {
        if ch == '0' {
            num0++
        } else if ch == '1' {
            num1++
        }
    }
    return num0, num1
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

一和零是二维费用 01 背包。

一个容量限制对应一个 DP 维度。

这里有 `0` 的数量和 `1` 的数量两个限制，
所以使用二维 `dp`。

空间优化后，
每个容量维度都要倒序遍历，
才能保证每个字符串只被使用一次。
