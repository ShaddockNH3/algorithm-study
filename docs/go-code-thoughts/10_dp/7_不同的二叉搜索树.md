# 96 不同的二叉搜索树

[题目链接](https://leetcode.cn/problems/unique-binary-search-trees/)

## 题目描述

给你一个整数 `n`，
求恰由 `n` 个节点组成，
且节点值从 `1` 到 `n` 互不相同的二叉搜索树有多少种。

返回满足题意的二叉搜索树种数。

## 核心思路

BST 的构造规则决定了这道题可以拆成左右子树问题。

如果选择 `i` 作为根节点：

- 比 `i` 小的数都必须在左子树。
- 比 `i` 大的数都必须在右子树。

于是问题被拆成两个独立子问题：

- 左子树有多少种构造方式。
- 右子树有多少种构造方式。

固定根节点后，
总方案数等于左子树方案数乘以右子树方案数。

## dp 数组定义

定义：

`dp[i]` 表示用 `i` 个连续且不同的节点，
能构成多少种不同的二叉搜索树。

最终要求：

```go
dp[n]
```

## 初始化

`dp[0] = 1`。

这表示空树也是一种情况。
如果没有这个初始化，
当某个根节点没有左子树或没有右子树时，
乘法递推就无法正确工作。

`dp[1] = 1` 会在循环中自然算出，
也可以显式理解为一个节点只有一种 BST。

## 状态转移

计算 `dp[i]` 时，
依次尝试每个节点作为根。

如果左子树有 `j` 个节点，
那么右子树有 `i - 1 - j` 个节点。

固定这种左右规模时，
方案数是：

```text
dp[j] * dp[i-1-j]
```

把所有 `j` 的情况累加：

```text
dp[i] += dp[j] * dp[i-1-j]
```

## 最终代码

```go
func numTrees(n int) int {
    dp := make([]int, n+1)
    dp[0] = 1

    for i := 1; i <= n; i++ {
        for j := 0; j < i; j++ {
            dp[i] += dp[j] * dp[i-1-j]
        }
    }

    return dp[n]
}
```

## 本篇总结

这道题是卡特兰数的典型应用。

重点是固定根节点后，
左子树和右子树是相互独立的组合。

所以固定根节点时用乘法原理，
枚举所有根节点时用加法原理。
