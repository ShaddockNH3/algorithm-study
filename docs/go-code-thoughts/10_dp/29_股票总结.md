# 股票买卖问题总结

股票系列的核心是动态规划状态设计。

大多数股票题都可以从两种基本状态出发：

- 持有股票。
- 不持有股票。

不同题目的复杂度，
来自交易次数、冷冻期、手续费等额外限制。

## 121 买卖股票的最佳时机

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

只能买卖一次。

### 121 贪心

维护历史最低价格 `minPrice`，
每天尝试以当天价格卖出。

```go
func maxProfit121Greedy(prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    minPrice := prices[0]
    maxProfit := 0

    for i := 1; i < len(prices); i++ {
        if prices[i] < minPrice {
            minPrice = prices[i]
        } else if prices[i]-minPrice > maxProfit {
            maxProfit = prices[i] - minPrice
        }
    }

    return maxProfit
}
```

### 121 DP

状态：

- `dp[i][0]`：第 `i` 天持有股票。
- `dp[i][1]`：第 `i` 天不持有股票。

只能买一次，
所以买入状态是 `-prices[i]`。

```go
func maxProfit121DP(prices []int) int {
    n := len(prices)
    if n == 0 {
        return 0
    }

    dp := make([][2]int, n)
    dp[0][0] = -prices[0]

    for i := 1; i < n; i++ {
        dp[i][0] = max(dp[i-1][0], -prices[i])
        dp[i][1] = max(dp[i-1][1], dp[i-1][0]+prices[i])
    }

    return dp[n-1][1]
}
```

## 122 买卖股票的最佳时机 II

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/)

可以无限次买卖。

### 122 贪心

累加所有上涨日的收益。

```go
func maxProfit122Greedy(prices []int) int {
    profit := 0

    for i := 1; i < len(prices); i++ {
        if prices[i] > prices[i-1] {
            profit += prices[i] - prices[i-1]
        }
    }

    return profit
}
```

### 122 DP

和 121 的区别在买入状态：

```text
dp[i][0] = max(dp[i-1][0], dp[i-1][1] - prices[i])
```

因为可以多次交易，
今天买入可以继承昨天不持有时已经获得的利润。

```go
func maxProfit122DP(prices []int) int {
    n := len(prices)
    if n == 0 {
        return 0
    }

    dp := make([][2]int, n)
    dp[0][0] = -prices[0]

    for i := 1; i < n; i++ {
        dp[i][0] = max(dp[i-1][0], dp[i-1][1]-prices[i])
        dp[i][1] = max(dp[i-1][1], dp[i-1][0]+prices[i])
    }

    return dp[n-1][1]
}
```

## 123 买卖股票的最佳时机 III

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iii/)

最多买卖两次。

需要 5 个状态：

- `0`：无操作。
- `1`：第一次买入。
- `2`：第一次卖出。
- `3`：第二次买入。
- `4`：第二次卖出。

```go
func maxProfit123DP(prices []int) int {
    n := len(prices)
    if n == 0 {
        return 0
    }

    dp := make([][5]int, n)
    dp[0][1] = -prices[0]
    dp[0][3] = -prices[0]

    for i := 1; i < n; i++ {
        dp[i][0] = dp[i-1][0]
        dp[i][1] = max(dp[i-1][1], dp[i-1][0]-prices[i])
        dp[i][2] = max(dp[i-1][2], dp[i-1][1]+prices[i])
        dp[i][3] = max(dp[i-1][3], dp[i-1][2]-prices[i])
        dp[i][4] = max(dp[i-1][4], dp[i-1][3]+prices[i])
    }

    return dp[n-1][4]
}
```

## 188 买卖股票的最佳时机 IV

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/)

最多买卖 `k` 次。

状态数组大小为 `2*k + 1`。

- 奇数状态表示买入。
- 偶数状态表示卖出。

```go
func maxProfit188DP(k int, prices []int) int {
    n := len(prices)
    if n == 0 {
        return 0
    }

    dp := make([][]int, n)
    for i := range dp {
        dp[i] = make([]int, 2*k+1)
    }

    for j := 1; j < 2*k; j += 2 {
        dp[0][j] = -prices[0]
    }

    for i := 1; i < n; i++ {
        for j := 0; j < 2*k-1; j += 2 {
            dp[i][j+1] = max(dp[i-1][j+1], dp[i-1][j]-prices[i])
            dp[i][j+2] = max(dp[i-1][j+2], dp[i-1][j+1]+prices[i])
        }
    }

    return dp[n-1][2*k]
}
```

## 309 含冷冻期

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-cooldown/)

无限次交易，
但卖出后有一天冷冻期。

需要把不持有状态拆得更细：

- `dp[i][0]`：持有股票。
- `dp[i][1]`：不持有，且不在冷冻期。
- `dp[i][2]`：不持有，且处于冷冻期，也就是今天刚卖出。

```go
func maxProfit309DP(prices []int) int {
    n := len(prices)
    if n == 0 {
        return 0
    }

    dp := make([][3]int, n)
    dp[0][0] = -prices[0]

    for i := 1; i < n; i++ {
        dp[i][0] = max(dp[i-1][0], dp[i-1][1]-prices[i])
        dp[i][1] = max(dp[i-1][1], dp[i-1][2])
        dp[i][2] = dp[i-1][0] + prices[i]
    }

    return max(dp[n-1][1], dp[n-1][2])
}
```

## 714 含手续费

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-transaction-fee/)

无限次交易，
但每次卖出时支付手续费。

和 122 基本一致，
只是在卖出转移时减去 `fee`。

```go
func maxProfit714DP(prices []int, fee int) int {
    n := len(prices)
    if n == 0 {
        return 0
    }

    dp := make([][2]int, n)
    dp[0][0] = -prices[0]

    for i := 1; i < n; i++ {
        dp[i][0] = max(dp[i-1][0], dp[i-1][1]-prices[i])
        dp[i][1] = max(dp[i-1][1], dp[i-1][0]+prices[i]-fee)
    }

    return dp[n-1][1]
}
```

## 股票系列总结

股票 DP 的核心是状态设计。

最基础的状态是持有和不持有。

如果限制交易次数，
就把买入、卖出次数展开成状态链。

如果有冷冻期，
就拆分不持有状态。

如果有手续费，
就在卖出状态转移中扣除费用。

```go
func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```
