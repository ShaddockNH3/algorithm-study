# 416 分割等和子集

[题目链接](https://leetcode.cn/problems/partition-equal-subset-sum/)

## 题目描述

给你一个只包含正整数的非空数组 `nums`。

判断是否可以将这个数组分割成两个子集，
使得两个子集的元素和相等。

## 问题转化

如果数组能分成两个和相等的子集，
数组总和 `sum` 必须是偶数。

如果 `sum` 是奇数，
直接返回 `false`。

如果 `sum` 是偶数，
每个子集的目标和就是：

```text
target = sum / 2
```

原问题可以转化为：

能否从 `nums` 中选出一些数，
使它们的和恰好等于 `target`。

## 背包建模

这是一个 01 背包问题。

- 背包容量是 `target`。
- 每个数字是一个物品。
- 物品重量是 `nums[i]`。
- 物品价值也可以看成 `nums[i]`。

`dp[j]` 表示容量为 `j` 时，
能装下的最大数值和。

如果最终 `dp[target] == target`，
说明正好能凑出目标和。

## 状态转移

对于每个数字 `nums[i]`：

```text
dp[j] = max(dp[j], dp[j-nums[i]] + nums[i])
```

因为每个数字只能用一次，
容量 `j` 要倒序遍历。

## 最终代码

```go
func canPartition(nums []int) bool {
    sum := 0
    for _, num := range nums {
        sum += num
    }

    if sum%2 == 1 {
        return false
    }

    target := sum / 2
    dp := make([]int, target+1)

    for i := 0; i < len(nums); i++ {
        for j := target; j >= nums[i]; j-- {
            dp[j] = max(dp[j], dp[j-nums[i]]+nums[i])
        }
    }

    return dp[target] == target
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

这道题的关键是把分成两个等和子集，
转化成能否凑出 `sum / 2`。

它本质上是 01 背包中的恰好装满问题。

只要背包容量为 `target` 时能装出的最大和等于 `target`，
就说明存在一个和为 `target` 的子集。
