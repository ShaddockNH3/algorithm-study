# 213 打家劫舍 II

[题目链接](https://leetcode.cn/problems/house-robber-ii/)

## 题目描述

这次房屋围成一圈。

每间房内有一定金额。
相邻房屋不能在同一晚被偷。

因为房屋成环，
第一间房和最后一间房也相邻。

返回不触发报警时能偷到的最高金额。

## 核心矛盾

和打家劫舍 I 相比，
新增的约束是：

```text
nums[0] 和 nums[len(nums)-1] 不能同时偷
```

因此可以把所有情况拆成两类。

第一类：
不偷最后一家。
可选范围是：

```text
nums[0 : len(nums)-1]
```

第二类：
不偷第一家。
可选范围是：

```text
nums[1 : len(nums)]
```

两个范围都变成了线性打家劫舍问题。

最终答案取两者最大值。

## 标准分解代码

```go
func rob(nums []int) int {
    if len(nums) == 0 {
        return 0
    }
    if len(nums) == 1 {
        return nums[0]
    }

    return max(robLinear(nums[:len(nums)-1]), robLinear(nums[1:]))
}

func robLinear(nums []int) int {
    pre, curr := 0, 0

    for _, num := range nums {
        temp := max(curr, num+pre)
        pre = curr
        curr = temp
    }

    return curr
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

环形打家劫舍的关键是破环成线。

由于首尾不能同时偷，
只需要分别计算：

- 不偷最后一家时的线性最优解。
- 不偷第一家时的线性最优解。

然后取最大值即可。

子问题完全复用打家劫舍 I 的线性求解器。
