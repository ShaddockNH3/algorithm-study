# 动态规划方法论

动态规划和贪心的一个重要区别是：

动态规划通常由前一个状态推导出当前状态。

贪心通常是在当前局部直接做出最优选择。

## DP 五步法

做动态规划题时，可以按下面五步拆解：

1. 确定 `dp` 数组以及下标的含义。
2. 确定递推公式。
3. 确定 `dp` 数组如何初始化。
4. 确定遍历顺序。
5. 举例推导 `dp` 数组。

## 调试检查

写完后可以问自己：

- 这道题是否举例推导了状态转移公式？
- 是否打印过 `dp` 数组？
- 打印出来的 `dp` 数组是否和预期一致？

## 本篇总结

DP 题最重要的是状态定义。

状态定义清楚之后，
递推公式、初始化和遍历顺序才有依据。

如果卡住，
先用小样例手动推一遍 `dp` 数组。
