# 343 整数拆分

[题目链接](https://leetcode.cn/problems/integer-break/)

## 题目描述

给定一个正整数 `n`，
将其拆分为 `k` 个正整数的和，
其中 `k >= 2`。

要求这些整数的乘积最大化。

返回可以获得的最大乘积。

示例：

```text
n = 2
2 = 1 + 1
最大乘积 = 1

n = 10
10 = 3 + 3 + 4
最大乘积 = 36
```

## dp 数组定义

定义：

`dp[i]` 表示将正整数 `i`
拆分成至少两个正整数之和后，
这些整数相乘能得到的最大乘积。

最终要求：

```go
dp[n]
```

## 状态转移

计算 `dp[i]` 时，
枚举第一刀切在哪里。

设第一刀切出 `j`，
剩下部分是 `i - j`。

对于 `i - j`，
有两种选择：

1. 不继续拆分，乘积是 `j * (i-j)`。
2. 继续拆分，乘积是 `j * dp[i-j]`。

所以固定 `j` 时，
当前最大值是：

```text
max(j * (i-j), j * dp[i-j])
```

遍历所有 `j`，
取最大值更新 `dp[i]`：

```text
dp[i] = max(dp[i], max(j * (i-j), j * dp[i-j]))
```

## 最终代码

```go
func integerBreak(n int) int {
    dp := make([]int, n+1)

    for i := 2; i <= n; i++ {
        for j := 1; j < i; j++ {
            dp[i] = max(dp[i], max(j*(i-j), j*dp[i-j]))
        }
    }

    return dp[n]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

整数拆分的关键是：
切出第一刀后，
剩下的部分到底继续拆分还是作为整体使用。

因此状态转移中要同时比较：

- `j * (i-j)`
- `j * dp[i-j]`

这也是本题和简单线性递推题的主要区别。
