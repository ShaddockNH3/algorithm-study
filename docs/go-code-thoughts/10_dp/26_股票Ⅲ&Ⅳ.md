# 123 和 188 股票 III 与股票 IV

[123. 买卖股票的最佳时机 III](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iii/)

[188. 买卖股票的最佳时机 IV](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/)

## 问题核心

当题目限制最多交易 `k` 次时，
只记录持有和不持有已经不够。

还必须记录当前已经处于第几次买入或卖出之后。

因此需要引入交易次数维度。

## 股票 III：最多交易两次

最多交易两次时，
可以设计 5 个状态：

- `dp[i][0]`：无操作。
- `dp[i][1]`：第一次买入后，持有股票。
- `dp[i][2]`：第一次卖出后，不持有股票。
- `dp[i][3]`：第二次买入后，持有股票。
- `dp[i][4]`：第二次卖出后，不持有股票。

状态转移链是：

```text
0 -> 1 -> 2 -> 3 -> 4
```

## 股票 III 代码

```go
func maxProfitTwoTransactions(prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    dp := make([][]int, len(prices))
    for i := range dp {
        dp[i] = make([]int, 5)
    }

    dp[0][1] = -prices[0]
    dp[0][3] = -prices[0]

    for i := 1; i < len(prices); i++ {
        dp[i][1] = max(dp[i-1][1], dp[i-1][0]-prices[i])
        dp[i][2] = max(dp[i-1][2], dp[i-1][1]+prices[i])
        dp[i][3] = max(dp[i-1][3], dp[i-1][2]-prices[i])
        dp[i][4] = max(dp[i-1][4], dp[i-1][3]+prices[i])
    }

    return dp[len(prices)-1][4]
}
```

## 股票 IV：最多交易 k 次

股票 III 可以推广到任意 `k` 次交易。

需要 `2*k + 1` 个状态：

- `j = 0`：无操作。
- `j` 为奇数：第 `(j+1)/2` 次买入后持有股票。
- `j` 为偶数且大于 0：第 `j/2` 次卖出后不持有股票。

初始化第 0 天时，
所有买入状态都可以看作当天买入：

```text
dp[0][1], dp[0][3], ..., dp[0][2*k-1] = -prices[0]
```

状态转移：

奇数状态是买入或继续持有：

```text
dp[i][j] = max(dp[i-1][j], dp[i-1][j-1] - prices[i])
```

偶数状态是卖出或继续不持有：

```text
dp[i][j] = max(dp[i-1][j], dp[i-1][j-1] + prices[i])
```

## 股票 IV 代码

```go
func maxProfit(k int, prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    dp := make([][]int, len(prices))
    for i := range dp {
        dp[i] = make([]int, 2*k+1)
    }

    for j := 1; j < 2*k; j += 2 {
        dp[0][j] = -prices[0]
    }

    for i := 1; i < len(prices); i++ {
        for j := 0; j <= 2*k; j++ {
            if j == 0 {
                dp[i][j] = dp[i-1][j]
            } else if j%2 == 1 {
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-1]-prices[i])
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-1]+prices[i])
            }
        }
    }

    return dp[len(prices)-1][2*k]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

股票 III 和 IV 的核心是状态升维。

交易次数限制不是额外的 if 判断，
而是要进入 DP 状态本身。

用奇数状态表示买入后持有，
用偶数状态表示卖出后不持有，
就可以把最多两次交易推广到最多 `k` 次交易。
