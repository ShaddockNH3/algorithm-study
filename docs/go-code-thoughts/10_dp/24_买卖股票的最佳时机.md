# 121 买卖股票的最佳时机

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

## 题目描述

给定一个数组 `prices`，
其中 `prices[i]` 表示股票第 `i` 天的价格。

只能选择某一天买入，
并在之后的某一天卖出。

最多完成一笔交易。

返回能够获得的最大利润。

## 解法一：贪心

一次交易的核心是：
在历史最低点买入，
在今天卖出，
看看利润是否最大。

遍历价格时维护两个变量：

- `low`：到目前为止见过的最低价格。
- `result`：到目前为止能获得的最大利润。

每天都更新最低买入价，
再尝试用今天价格卖出。

```go
func maxProfitGreedy(prices []int) int {
    low := 10001
    result := 0

    for _, price := range prices {
        low = min(low, price)
        result = max(result, price-low)
    }

    return result
}
```

这个写法时间复杂度 `O(n)`，
空间复杂度 `O(1)`，
是本题最直接的解法。

## 解法二：动态规划

股票系列可以用统一 DP 模板。

定义每天结束后的两种状态：

- `dp[i][0]`：第 `i` 天结束时持有股票的最大现金。
- `dp[i][1]`：第 `i` 天结束时不持有股票的最大现金。

对于只能买卖一次的题目：

持有股票：

```text
dp[i][0] = max(dp[i-1][0], -prices[i])
```

这里今天买入时只能从 0 现金开始，
因为最多只允许一次交易。

不持有股票：

```text
dp[i][1] = max(dp[i-1][1], dp[i-1][0] + prices[i])
```

注意卖出时要使用昨天持有股票的状态 `dp[i-1][0]`。

## DP 代码

```go
func maxProfitDP(prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    dp := make([][]int, len(prices))
    for i := range dp {
        dp[i] = make([]int, 2)
    }

    dp[0][0] = -prices[0]
    dp[0][1] = 0

    for i := 1; i < len(prices); i++ {
        dp[i][0] = max(dp[i-1][0], -prices[i])
        dp[i][1] = max(dp[i-1][1], prices[i]+dp[i-1][0])
    }

    return dp[len(prices)-1][1]
}
```

## 空间优化

因为只依赖前一天状态，
可以用两个变量：

- `hav`：前一天持有股票的最大现金。
- `no`：前一天不持有股票的最大现金。

计算 `no` 时需要旧的 `hav`。

```go
func maxProfit(prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    hav := -prices[0]
    no := 0

    for i := 1; i < len(prices); i++ {
        no = max(no, prices[i]+hav)
        hav = max(hav, -prices[i])
    }

    return no
}

func min(a, b int) int {
    if a < b {
        return a
    }
    return b
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

本题贪心解法最简洁：
维护历史最低价格和最大利润。

DP 写法看起来更重，
但它是股票系列通用模板的起点。

一次交易时，
买入状态只能来自 `-prices[i]`，
不能继承之前不持有状态的利润。
