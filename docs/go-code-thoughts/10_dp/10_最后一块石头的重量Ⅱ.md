# 1049 最后一块石头的重量 II

[题目链接](https://leetcode.cn/problems/last-stone-weight-ii/)

## 题目描述

有一堆石头，
用整数数组 `stones` 表示。

每一回合选出任意两块石头粉碎。
如果石头重量分别为 `x` 和 `y`，
且 `x <= y`：

- 如果 `x == y`，两块石头都被粉碎。
- 如果 `x != y`，重量为 `x` 的石头被粉碎，`y` 变成 `y - x`。

最后最多剩下一块石头。
返回这块石头最小的可能重量。
如果没有石头剩下，返回 `0`。

## 问题转化

石头碰撞可以理解为给每块石头分配正负号。

最后结果就是所有石头带符号求和后的绝对值。

也可以理解为：
把石头分成两堆，
让两堆重量尽可能接近。

两堆越接近，
最后剩下的差值越小。

## 背包建模

先计算总重量 `stoneSum`。

目标是找到一堆石头，
它的总重量尽可能接近 `stoneSum / 2`。

这就是 01 背包：

- 背包容量是 `stoneSum / 2`。
- 物品是每块石头。
- 重量和价值都是 `stones[i]`。

`dp[j]` 表示容量为 `j` 时，
能装下的最大石头重量。

## 最终结果

设 `dp[target]` 是能凑出的最接近一半的重量。

另一堆重量是：

```text
stoneSum - dp[target]
```

两堆差值是：

```text
stoneSum - dp[target] - dp[target]
```

也就是：

```text
stoneSum - 2 * dp[target]
```

## 最终代码

```go
func lastStoneWeightII(stones []int) int {
    stoneSum := 0
    for _, stone := range stones {
        stoneSum += stone
    }

    target := stoneSum / 2
    dp := make([]int, target+1)

    for i := 0; i < len(stones); i++ {
        for j := target; j >= stones[i]; j-- {
            dp[j] = max(dp[j], dp[j-stones[i]]+stones[i])
        }
    }

    return stoneSum - 2*dp[target]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

这道题不是直接模拟石头碰撞，
而是转化成两堆石头重量尽可能接近。

找到不超过总重量一半的最大可凑重量后，
最终答案就是：

```text
stoneSum - 2 * dp[target]
```
