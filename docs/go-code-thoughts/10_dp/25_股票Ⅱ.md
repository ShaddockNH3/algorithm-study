# 122 买卖股票的最佳时机 II

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/)

## 题目描述

给定数组 `prices`，
其中 `prices[i]` 表示股票第 `i` 天的价格。

可以完成多次交易，
但同一时间最多只能持有一股股票。

返回能获得的最大利润。

## 解法一：贪心

因为可以无限次交易，
只要后一日价格比前一日高，
就可以吃下这段上涨收益。

连续上涨的总利润可以拆成每天的正收益之和。

```go
func maxProfitGreedy(prices []int) int {
    result := 0

    for i := 1; i < len(prices); i++ {
        if prices[i] > prices[i-1] {
            result += prices[i] - prices[i-1]
        }
    }

    return result
}
```

## 解法二：动态规划

状态仍然是：

- `dp[i][0]`：第 `i` 天结束时持有股票的最大现金。
- `dp[i][1]`：第 `i` 天结束时不持有股票的最大现金。

和 121 题不同的是，
这里可以多次交易。

所以今天买入时，
可以使用昨天不持有股票时已经获得的现金：

```text
dp[i][0] = max(dp[i-1][0], dp[i-1][1] - prices[i])
```

今天不持有时：

```text
dp[i][1] = max(dp[i-1][1], dp[i-1][0] + prices[i])
```

## DP 代码

```go
func maxProfitDP(prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    dp := make([][]int, len(prices))
    for i := range dp {
        dp[i] = make([]int, 2)
    }

    dp[0][0] = -prices[0]
    dp[0][1] = 0

    for i := 1; i < len(prices); i++ {
        dp[i][1] = max(dp[i-1][1], prices[i]+dp[i-1][0])
        dp[i][0] = max(dp[i-1][0], dp[i-1][1]-prices[i])
    }

    return dp[len(prices)-1][1]
}
```

## 空间优化

计算今天持有状态时，
要使用昨天不持有的旧值。

所以先保存 `noOld`。

```go
func maxProfit(prices []int) int {
    if len(prices) == 0 {
        return 0
    }

    hav := -prices[0]
    no := 0

    for i := 1; i < len(prices); i++ {
        noOld := no
        no = max(no, prices[i]+hav)
        hav = max(hav, noOld-prices[i])
    }

    return no
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 本篇总结

股票 II 和股票 I 的区别在于可以买卖多次。

DP 中最关键的变化是：

```text
dp[i][0] = max(dp[i-1][0], dp[i-1][1] - prices[i])
```

也就是今天买入可以继承之前已经赚到的利润。

贪心法则是抓住所有上涨段，
更适合本题本身。
