# 滑动窗口最大值：单调队列维护窗口最大值

滑动窗口最大值是单调队列的经典应用。

目标是在窗口不断右移时，快速得到当前窗口中的最大值。

## 239 滑动窗口最大值

[题目链接](https://leetcode.cn/problems/sliding-window-maximum/)

### 239 题意

给定整数数组 `nums` 和窗口大小 `k`。

窗口从数组最左侧移动到最右侧，每次向右移动一位。
返回每个窗口中的最大值。

## 为什么需要单调队列

暴力法每次窗口移动后，都遍历窗口内的 `k` 个元素找最大值。

时间复杂度是：

```text
O(n * k)
```

当 `n` 和 `k` 都较大时会超时。

单调队列可以把整体时间复杂度降到 `O(n)`。

## 单调队列规则

单调队列维护一个从队头到队尾单调递减的队列。

它有三个关键操作：

1. `Front`：
   队头元素始终是当前窗口最大值。
2. `Push`：
   新元素入队前，先从队尾弹出所有比它小的元素。
3. `Pop`：
   窗口左侧元素离开时，只有它等于队头元素，才弹出队头。

为什么 `Push` 要弹出更小元素：

如果新元素 `x` 比队尾元素更大，
那么队尾元素既比 `x` 小，又比 `x` 更早离开窗口。
它之后不可能再成为最大值，可以直接移除。

为什么 `Pop` 只检查队头：

只有队头元素才可能是当前最大值。
如果滑出窗口的值不是队头，它即使离开，也不会影响当前最大值。

## 239 代码

```go
type MyQueue struct {
    deque []int
}

func NewMyQueue() *MyQueue {
    return &MyQueue{
        deque: make([]int, 0),
    }
}

func (m *MyQueue) Front() int {
    return m.deque[0]
}

func (m *MyQueue) Back() int {
    return m.deque[len(m.deque)-1]
}

func (m *MyQueue) Empty() bool {
    return len(m.deque) == 0
}

func (m *MyQueue) Push(x int) {
    for !m.Empty() && x > m.Back() {
        m.deque = m.deque[:len(m.deque)-1]
    }
    m.deque = append(m.deque, x)
}

func (m *MyQueue) Pop(x int) {
    if !m.Empty() && x == m.Front() {
        m.deque = m.deque[1:]
    }
}

func maxSlidingWindow(nums []int, k int) []int {
    ans := []int{}
    deque := NewMyQueue()

    for i := 0; i < k; i++ {
        deque.Push(nums[i])
    }
    ans = append(ans, deque.Front())

    for i := k; i < len(nums); i++ {
        deque.Pop(nums[i-k])
        deque.Push(nums[i])
        ans = append(ans, deque.Front())
    }

    return ans
}
```

## 示例过程

以：

```text
nums = [1,3,-1,-3,5,3,6,7], k = 3
```

初始窗口 `[1,3,-1]`：

```text
Push(1)  -> [1]
Push(3)  -> [3]
Push(-1) -> [3, -1]
最大值：3
```

窗口滑动到 `[3,-1,-3]`：

```text
Pop(1)   -> [3, -1]
Push(-3) -> [3, -1, -3]
最大值：3
```

窗口滑动到 `[-1,-3,5]`：

```text
Pop(3)  -> [-1, -3]
Push(5) -> [5]
最大值：5
```

单调队列始终让队头保存当前窗口最大值。

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(k)`

每个元素最多入队一次、出队一次，
所以整体是线性复杂度。

## 本题总结

单调队列适合维护滑动窗口中的最值。

这道题的关键是：

- 队列从头到尾保持单调递减。
- 新元素入队时，删除队尾所有更小元素。
- 旧元素离开窗口时，只有它等于队头才弹出。

这样每次窗口移动后，队头就是当前窗口最大值。
