# 用队列实现栈：旋转队列取得栈顶

用队列实现栈，本质上是在用先进先出的结构模拟后进先出。

这篇采用单队列实现。
核心思路是：在 `Pop` 时旋转队列，把队尾元素移动到队头。

## 225 用队列实现栈

[题目链接](https://leetcode.cn/problems/implement-stack-using-queues/)

### 225 题意

仅使用队列实现一个后进先出的栈，并支持以下操作：

- `push(x)`：将元素压入栈顶。
- `pop()`：移除并返回栈顶元素。
- `top()`：返回栈顶元素。
- `empty()`：判断栈是否为空。

## 单队列思路

队列的特性是先进先出。

栈的特性是后进先出。

新元素入队后，会出现在队尾。
而栈的 `Pop` 需要取出最后进入的元素，也就是队尾元素。

因此，在执行 `Pop` 时，可以把队列前面的 `n-1` 个元素依次出队
再入队。这样原来的队尾元素就会被旋转到队头。

## 225 代码

```go
type MyStack struct {
    queue []int
}

func Constructor() MyStack {
    return MyStack{
        queue: make([]int, 0),
    }
}

func (this *MyStack) Push(x int) {
    this.queue = append(this.queue, x)
}

func (this *MyStack) Pop() int {
    n := len(this.queue)

    for i := 0; i < n-1; i++ {
        val := this.queue[0]
        this.queue = this.queue[1:]
        this.queue = append(this.queue, val)
    }

    val := this.queue[0]
    this.queue = this.queue[1:]
    return val
}

func (this *MyStack) Top() int {
    return this.queue[len(this.queue)-1]
}

func (this *MyStack) Empty() bool {
    return len(this.queue) == 0
}
```

## Pop 过程

假设队列当前为：

```text
[1, 2, 3, 4]
```

其中 `4` 是最后进入的元素，也是栈顶。

执行 `Pop` 时，将前 `n-1` 个元素移到队尾：

```text
[1, 2, 3, 4] -> [2, 3, 4, 1]
[2, 3, 4, 1] -> [3, 4, 1, 2]
[3, 4, 1, 2] -> [4, 1, 2, 3]
```

此时 `4` 已经来到队头。
再执行一次出队，就得到了栈顶元素。

## Top 操作

`Top` 只是查看栈顶，不需要删除元素。

在这个实现中，新元素始终追加到切片末尾。
因此队尾元素就是当前栈顶，可以直接返回：

```go
return this.queue[len(this.queue)-1]
```

## 复杂度

- `Push`：`O(1)`
- `Pop`：`O(n)`
- `Top`：`O(1)`
- `Empty`：`O(1)`

这套实现把主要成本放在 `Pop` 上，
换来更简单的 `Push` 和 `Top`。

## 本题总结

单队列实现栈的关键是旋转队列。

当需要弹出栈顶时，先把队头的 `n-1` 个元素移到队尾，
使原队尾元素变成队头，再弹出它。
