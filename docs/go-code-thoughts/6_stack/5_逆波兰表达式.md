# 逆波兰表达式求值：遇到运算符就弹栈计算

逆波兰表达式也叫后缀表达式。
它的特点是运算符位于操作数后面。

这种表达式非常适合用栈求值。

## 150 逆波兰表达式求值

[题目链接](https://leetcode.cn/problems/evaluate-reverse-polish-notation/)

### 150 题意

给定一个字符串数组 `tokens`，
表示一个根据逆波兰表示法表示的算术表达式。

计算该表达式，并返回整数结果。

示例：

```text
输入：tokens = ["2", "1", "+", "3", "*"]
输出：9
解释：((2 + 1) * 3) = 9
```

## 逆波兰表达式

逆波兰表达式的运算符在操作数之后。

例如：

```text
3 + 4 -> 3 4 +
(2 + 1) * 3 -> 2 1 + 3 *
```

计算规则是从左到右遍历：

1. 遇到数字，先入栈。
2. 遇到运算符，从栈顶弹出最近两个数字。
3. 用这两个数字计算。
4. 把计算结果重新压入栈。
5. 遍历结束后，栈中唯一的数字就是答案。

## 150 代码

```go
import "strconv"

func evalRPN(tokens []string) int {
    stack := []int{}

    for _, token := range tokens {
        num, err := strconv.Atoi(token)

        if err == nil {
            stack = append(stack, num)
        } else {
            n := len(stack)
            num1, num2 := stack[n-2], stack[n-1]
            stack = stack[:n-2]

            switch token {
            case "+":
                stack = append(stack, num1+num2)
            case "-":
                stack = append(stack, num1-num2)
            case "*":
                stack = append(stack, num1*num2)
            case "/":
                stack = append(stack, num1/num2)
            }
        }
    }

    return stack[0]
}
```

## 运算顺序

弹出两个数后，顺序不能反。

代码中：

```go
num1, num2 := stack[n-2], stack[n-1]
```

其中：

- `num1` 是左操作数。
- `num2` 是右操作数。

因此减法和除法要写成：

```go
num1 - num2
num1 / num2
```

不能写反。

## 示例过程

以 `["2", "1", "+", "3", "*"]` 为例：

遇到 `2`：

```text
stack = [2]
```

遇到 `1`：

```text
stack = [2, 1]
```

遇到 `+`：

```text
2 + 1 = 3
stack = [3]
```

遇到 `3`：

```text
stack = [3, 3]
```

遇到 `*`：

```text
3 * 3 = 9
stack = [9]
```

最终答案是 `9`。

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 本题总结

逆波兰表达式适合用栈，是因为它总是在运算符出现时，
立刻使用最近的两个数字计算。

最近的两个数字正好位于栈顶。
计算完成后再把结果压回栈，继续参与后续计算。
