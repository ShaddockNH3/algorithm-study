桌子上有1堆糖果，一共有n颗,两人轮流取糖果。规定每次最少拿1颗，最多拿m颗，最后拿完的人获胜。(两个人都很聪明，都懂得使用最优策略)

### 输入格式:

输入两个整数n，m（0<m,n<10000000)。

### 输出格式:

先拿的人获胜输出"First win"，否则输出"Second win"。

### 输入样例1:

```in
7 5
```

### 输出样例1:

```out
First win
```

### 输入样例2:

```in
6 5
```

### 输出样例2:

```out
Second win
```

代码长度限制

16 KB

时间限制

400 ms

内存限制

64 MB

栈限制

8192 KB

```
#include <stdio.h>
int main()
{
    int n,m;
    scanf("%d%d",&n,&m);
    if(n%(m+1)==0){
        printf("Second win");
    }
    else{
        printf("First win");
    }
    return 0;
}
```
