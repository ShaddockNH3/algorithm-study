本质上是求解以下这个问题：

$max(prices[j]−prices[i])$

那么就需要找到局部最大和局部最小，也就是要让后面的数尽可能大而前面的数尽可能小。所以其实不难，代码如下：

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int len=prices.size();
        int min_price=prices[0];
        int max_profit=0;

        for(int i=1;i<len;i++){
            if(min_price>prices[i]){
                min_price=prices[i];
            }
            else{
                max_profit=max(max_profit,prices[i]-min_price);
            }
        }

        return max_profit;
    }
};
```
