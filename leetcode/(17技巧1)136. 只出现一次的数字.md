[136. 只出现一次的数字 - 力扣（LeetCode）](https://leetcode.cn/problems/single-number/description/?envType=study-plan-v2&envId=top-100-liked)

很简单，全部xor一遍

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int now=nums[0];
        for(int i=1;i<nums.size();i++){
            now^=nums[i];
        }
        return now;
    }
};
```
